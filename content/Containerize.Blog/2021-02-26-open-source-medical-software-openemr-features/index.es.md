## ¿Qué es Openemr   {#Intro}
[OpenEmr][6] es una solución multilingüe de atención médica de código abierto. Ofrece seguridad de datos con técnicas de cifrado. Esta solución de salud gratuita facilita a sus usuarios para administrar los registros médicos de los pacientes y el stock de medicina. Además, hay muchas características, como programar citas de pacientes, recordatorios e integración de laboratorio. Sin embargo, este software médico de código abierto tiene un módulo de facturación integrado muy potente para administrar transacciones y facturas financieras. Openemr es multilingüe y brinda soporte para casi más de 100 idiomas.
Este software de facturación médica basado en la web está a sí mismo y proporciona una interfaz lógica y fácil de usar. Además, la prescripción electrónica es un módulo muy poderoso de esta herramienta gratuita. Sobre todo, existen capacidades de búsqueda ricas en las que los administradores de software médico pueden realizar búsquedas en módulos primarios. Es altamente extensible y ofrece soporte para la integración de aplicaciones de terceros. OpenEMR está escrito en PHP y está completamente documentado con respecto al desarrollo y la implementación. Además, esta solución de salud médica es fácil de configurar y requiere requisitos simples como Apache, PHP y MySQL.

## OpenEmr características   {#Feature}
En esta sección, exploraremos algunos de los módulos principales ofrecidos por este **Software de atención médica personalizada** .
Puede experimentar OpenEMR usted mismo siguiendo este [enlace][7]. Una vez que abra este enlace, aterrizará en una página de inicio de sesión. Puede usar las siguientes credenciales para iniciar sesión como administrador.
Nombre de usuario: administrador
Contraseña: pasar
Tras un inicio de sesión exitoso, verá el tablero de administrador como se muestra en la imagen a continuación.
En primer lugar, si entras en la configuración, puedes hacer varias acciones, como cambiar la apariencia del tablero, los informes y la facturación. Hay una sección llamada "Centro de mensajes" que se utiliza para intercambiar mensajes entre los usuarios con respecto a los pacientes. Hay desplegables donde los usuarios pueden seleccionar cualquier usuario y pacientes con la disposición para borrar los valores seleccionados. Además, puede encontrar un área de texto para escribir mensajes siempre que sea necesario. Además, hay una barra de menú en la parte superior del tablero con varios elementos de menú, como Calander, tablero de flujo, tablero de retiro, mensajes, pacientes, tarifas, módulos, procedimientos, administración, informes y algunos más.

{{< figure align=center src="images/Screen-Shot-2021-02-25-at-6.22.35-PM-1024x577.png" alt="Software de atención médica personalizada">}}

Además, también permite a los usuarios configurar y cambiar la configuración del calendario. En el módulo del paciente, los usuarios pueden crear registros de nuevos pacientes, visitas al paciente, tipo de visita y también los usuarios pueden importar datos de pacientes.

## facturación y otras características
En la imagen de arriba, puede ver la barra de menú superior y el calendario en el lado izquierdo. Sin embargo, puede cambiar entre los elementos del menú. Bueno, en el centro del tablero puede ver las citas programadas de los pacientes. Puede ver todos los detalles médicos/personales del paciente haciendo clic en el nombre. De hecho, los usuarios pueden aprovechar un poderoso módulo de facturación que también admite pagos por lotes. Además, existe un sistema totalmente integrado para los informes donde los usuarios pueden encontrar informes de pacientes generados por varias clínicas y laboratorios. Esta solución de salud médica ofrece una provisión para instalar y administrar módulos necesarios para la funcionalidad necesaria.
Hay un módulo completo para agregar/controlar a los pacientes de la interfaz de usuario. La ventana emergente del paciente Agregar le permite insertar detalles completos del paciente, como DOB, estado civil, detalles de facturación, etc.
