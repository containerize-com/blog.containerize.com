## ¿Qué es la tecnología SSO? {#what}

[Sign-On][6] es un software de identificación y autenticación**que le permite iniciar sesión utilizando sitios de terceros. El software SSO actúa como un proveedor de identidad para que pueda iniciar sesión en varios sitios con un solo conjunto de nombres de usuario/contraseñas. Sin embargo, es un mecanismo de inicio de sesión único y el sistema SSO asegura la autenticidad de los usuarios.
Además, este sistema se ha utilizado ampliamente a nivel pequeño y empresarial para administrar las identidades de los usuarios en múltiples aplicaciones. Hay muchos software de inicio de sesión único de código abierto que utilizan protocolos de autorización populares como OpenID OpenID, SAML y OAUTH.

##  **¿Cómo funciona el inicio de sesión único (SSO)?**  {#features}

El mecanismo de trabajo de  **Sign-On de código abierto**  es recto. El usuario ingresa credenciales en el formulario de inicio de sesión, el usuario se moverá al completar con éxito la autenticación SSO. Ahora, cada vez que un usuario visita cualquier sitio integrado con cualquier solución SSO, el sistema verificará que cualquiera de los usuarios ya haya iniciado sesión o no. Además, si el usuario ya ha iniciado sesión, el servidor emitirá un rastreador/token y este token/rastreador se utilizará para asegurarse de que su autenticación en múltiples aplicaciones. Junto con las capacidades de SSO, también proporciona funciones de gestión y seguridad de los usuarios como se menciona a continuación.
  * Fácil de usar
  * Identidad federada
  * Farricz
  * Protocolo OAuth V2
  * Gestión de usuarios
  * Protocolo de Connect OpenID y OpenID
  * 2f Autenticación
  * Flexible
  * Administración de contraseñas
  * Autorización de autenticación

## Casos de uso comercial del sistema SSO {#business}

En esta sección, observaremos algunos casos centrales de uso comercial de  **Sign-On.** Por lo tanto, un sistema de autenticación centralizado*  ***proporciona una mejor gestión de usuarios, seguridad y flexibilidad. Aquí están las siguientes aplicaciones comerciales del sistema**  **SSO** .** 
 **Reduce la fatiga de la contraseña** : El software SSO reduce el tiempo y la molestia tanto para organizaciones como para empleados. Los usuarios no necesitan ingresar credenciales una y otra vez. Además, hay menos carga para los equipos de soporte en los que no necesitan preocuparse por la administración de contraseña/nombre de usuario en múltiples aplicaciones.
 **Autenticación centralizada** : Lo mejor que se le ocurre el sistema **SSO** es un punto central de autenticación.
 **Seguridad** : Además, este sistema de software ofrece una tensa seguridad con la ayuda de la autorización de dos factores junto con otros protocolos de seguridad.
 **Flexible** : Los administradores de la compañía pueden vincular a los empleados y recursos con un solo conjunto de credenciales. Además, los administradores pueden personalizar de acuerdo con las necesidades, ya que pueden automatizar la administración de acceso.
 **Protección de identidad** : Sobre todo, el software de inicio de sesión único utiliza la autenticación de dos factores y multifactor para asegurarse de que la integridad y la seguridad de los datos confidenciales.
