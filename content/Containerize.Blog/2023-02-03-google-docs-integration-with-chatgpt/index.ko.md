## 반복적 인 행동을 현대화하려고합니까? 다음은 Google Docs Automation을위한 광범위한 기능을 제공하는 AI 구동 Openai GPT-3 NLP 모델입니다.

{{< figure align=center src="images/Google-Docs-Integration-with-ChatGPT-OpenAI-GPT-3-1.png" alt="ChatGpt와의 Google Docs 통합">}}


## 개요
최근에 출시 된 [Chatgpt][1]는 기술 세계에서 유행어가되었습니다. 어떤 사람들은 호의적 이며이 **ai 챗봇  **생산성과 투명성에 대한 위협이라고 생각합니다. [Containerize.com][2]는이 OpenAI GPT-3 모델의 사용 및 통합에 대해 쓰고 있습니다. 실제로 [Google Docs][3]는 쓰기 및 데이터 표현을위한 훌륭한 도구입니다. 또 다른 한편으로, Chatgpt의 출현은 대화 [AI][4] 도구입니다. 이 AI 도구는 기계 학습을 기반으로하며 [OpenAI][5]에 의해 개발되었습니다. 그러나이 블로그 게시물은 **  chatgpt 및 google **에 관한 것입니다. 따라서 chatgpt **와의 Google Docs 통합을 달성하는 방법을 배울 것입니다. 의심 할 여지없이 Google 문서에 chatgpt 애드온을 설치하여  **Google Docs Automation**  을 가져올 수 있습니다.
이 블로그 게시물에서 다음 사항을 살펴 보겠습니다.
  *[**OpenAI API 키를 얻는 방법?** ][6]
  * **[ChatGpt와 Google 문서 통합을 해보자][7]**
  *[**chatgpt 및 google 사용 시작  ****  문서** ][8]

## OpenAI API 키를 얻는 방법은 무엇입니까?   {#how-obtain-the-openai-api-key}
약간의 소개를 제공하기 위해 OpenAi는 인공 지능 도구의 연구 및 개발에 중점을 둔 회사입니다. Chatgpt는 OpenAI에서도 개발 및 출시되는 반면 OpenAi는 일련의 GPT 모델을 개발했습니다.
먼저 OpenAi의 [플랫폼][9]에서 계정을 생성 한 다음 로그인해야합니다. 로그인하면 로그인 한 후에는 API 키 페이지에 착륙하여 버튼을 클릭하여 API 키를 생성 할 수 있습니다. 키”아래 그림과 같이.

{{< figure align=center src="images/openai-api-key-1.gif" alt="Openai API 키">}}

새로 생성 된 OpenAI API 키를 복사하여 Google Docs 통합에 사용할 수 있도록 저장하십시오.

## chatgpt   {#lets-do-google-docs-integration-with-Chatgpt}와 Google 문서 통합을 해보자.
이 대화식 AI 도구의 설치 절차는 매우 쉽습니다. “Extensions-Add-ons-> Add-ong get”을 클릭하면 Google Workspace Marketplace 페이지가 표시됩니다. 이제 검색 표시 줄에서 chatgpt를 쓰고“시트 및 문서의 Chatgpt”옵션을 선택하고 첫 번째 옵션을 클릭하고 아래 이미지에 표시된대로 설치하십시오.

{{< figure align=center src="images/chatgpt-and-google-1.gif" alt="Chatgpt와 Google">}}

Google 계정에 액세스 할 수있는 권한을 요청 하므로이 단계를 수행하여 설치를 완료하십시오. 그런 다음 아래 이미지에 표시된대로 ChatGpt와 Google Docs 통합을 활성화 할 수 있도록 API 키를 설정하려면 "시트 및 문서 및 Docs-> Set API 키 설정"을 클릭하십시오.

{{< figure align=center src="images/conversational-ai-tool.gif" alt="대화식 AI 도구">}}

마지막으로 OpenAI GPT-3이 Google 문서에 설치되었으며 Google Docs Automation을 활용할 수 있습니다.

## chatgpt 및 Google Docs 사용 시작   {#start-using-chatgpt and-google}
이 Google Docs Automation을 사용해야 할 때입니다. 이 목적으로, "Extensions-> 앱 스크립트"를 클릭하여 새 탭이 열립니다. 이제 다음 코드를 코드 편집기에 복사하여 붙여 넣습니다.
{{< gist mustafabutt-dev 997afd9cc9f7285b055e053ad95ff298 >}}
위의 코어에서 "xxxxxxxxxxxxxxx"를 OpenAI API 키로 바꾸십시오. 다음으로 파일을 저장하고 코드 편집기의 상단 중앙에 위치한 "실행"을 클릭하고 창을 닫습니다.
Google Docs 창으로 돌아 가면 메뉴 표시 줄에 새 메뉴 항목 "Chatgpt"가 표시됩니다. 요구 사항에 따라 코드를 수정할 수 있지만이 예에서는 이미지 생성, 태그 라인 및 아래 이미지에 표시된 동의어와 같은 다양한 기능이 있습니다.

{{< figure align=center src="images/Google-Docs-Integration-.gif" alt="ChatGpt와의 Google Docs 통합">}}

위의 예에서, 우리는 단어를 작성하고 Chatgpt에게 태그 라인, 이미지 및 동의어를 생성하도록 요청했으며 작업을 성공적으로 수행했습니다.
