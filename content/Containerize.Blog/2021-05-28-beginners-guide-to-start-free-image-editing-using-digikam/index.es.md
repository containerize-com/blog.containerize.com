## ¿Qué es Digikam? {#digikam}

Para cualquier fotógrafo digital que tenga un software que ayude al flujo de trabajo de la fotografía y lo hará sin problemas, puede marcar una gran diferencia no solo la calidad de sus fotos sino también del proceso de edición. Digikam es un administrador de flujo de trabajo de fotografía digital que permite a los fotógrafos no solo clasificar los archivos por separado a granel, sino también aplicar muchos efectos de edición. También puede cargar directamente las imágenes sin procesar y convertirlas en DNG sin pérdida de calidad, y admite muchos modelos de cámara digital. Como editor de imágenes de código abierto, también puede almacenar archivos localmente o en un servidor remoto. Digikam proporciona soporte de bases de datos múltiples que incluyen SQL, MySQL y Mariadb.
Digikam tiene una interfaz fácil de usar. Le permite conectarse a su cámara o imágenes de vista previa. Su diseño es muy intuitivo y fácil de entender. A la izquierda hay controles para ver fotos y a la derecha hay controles para efectos y filtros. Dgikam ha sido esencialmente una aplicación completa y estable durante bastante tiempo, por lo que será una buena experiencia para usted.

## ¿Cómo instalar Digikam? {#install}

[Digikam][1] admite múltiples sistemas operativos que incluyen Windows, Mac OS y Linux. Como software de edición de imágenes gratuitos, la última versión de Digikam es bastante estable y está lista para instalarse para su sistema operativo preferido.

  * Ahora ejecute los dos comandos mencionados a continuación para instalar Digikam:
```
sudo apt update
```
```
sudo apt install digikam
```
  * Y ha terminado, ahora puede iniciar la aplicación y configurar el comportamiento del archivo de acuerdo con sus necesidades.
Es importante tener en cuenta que el primer lanzamiento de Digikam será lento en cuanto al rendimiento, ya que Digikam analizará su colección de fotos y creará información para ello. Esto puede llevar un tiempo, por lo que es mejor dejarlo correr durante la noche si tiene una gran colección y comienza a usar al día siguiente.
