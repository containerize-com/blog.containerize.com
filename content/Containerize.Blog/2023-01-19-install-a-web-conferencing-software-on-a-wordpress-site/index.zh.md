##**什么是bigbluebutton？**{#什么 -  bigbluebutton}
[BigBlueButton][6]是一个开源免费**视频协作软件**，具有大量的企业级功能。该开源软件非常可扩展，并与WordPress，[Redmine][7]，[Schoology][8]，[Drupal][9]和Canvas等许多流行应用程序提供集成。此外，它还提供了共享的白板，发送公共/私人消息，高清音频，电子手工升级以及丰富的演示工具。此外，**BigBlueButton**得到了HTML5的支持，并且在所有流行的Web浏览器中都可以很好地工作。
此外，这个**网络会议软件**拥有一个庞大而活着的开发人员社区。它的[GitHub Repo][10]上有非常好的统计数据，其中有7.8k星和5.8k叉。 BigBluebutton用JavaScript编写，以及Java和Ruby等其他语言的输入。实际上，开发人员可以找到有关开发，部署和使用的全面文件。因此，您可以轻松地在本地或云服务器上设置此自托管软件。

##**如何在WordPress中安装BigBlueButton插件？
本节将演示如何**在WordPress网站上安装Web会议软件**。此外，我们还将更多地探索BigBlueButton插件。在进一步移动之前，请确保您已经在计算机上安装了WordPress并将业务网站启动和运行。
请访问此[链接][2]，如果您没有安装WordPress，则可以找到完整的安装说明。
让我们开始在WordPress上的BigBlueButton [插件][13]的安装过程。
转到WordPress仪表板，然后打开“插件”选项，然后按“添加新”。在搜索栏中写入BigBlueButton，您会发现插件在搜索中显示第一个位置，如下图所示：
{{_LINE_29_}}
现在，点击“立即安装”按钮以安装此**Web会议软件**并点击“激活”以激活插件。激活活力后，您可以开始使用此视频会议软件。向前迈进，您可以在窗口的左侧面板上看到菜单项“房间”的新添加，如下图所示。

{{< figure align=center src="images/bigbluebutton-plugin-1024x516.png" alt="BigBlueButton插件">}}

因此，现在，您可以通过创建和出版室来创建会议室，在那里您可以与伴侣进行视频会议。此外，您可以通过更改背景颜色并上传品牌徽标来自定义房间的外观。在下图中，您可以看到我们创建了一个会议，并且会议室的用户界面非常丰富，具有实时聊天和共享笔记之类的功能。
