## CMS、通信、カスタマーサポート、マーケティングおよび財務運営など、ビジネスのすべてのセグメント向けの無料およびオープンソースソフトウェアのリスト。

{{< figure align=center src="images/zuMM6v9k-1024x536.png" alt="無料のオープンソースソフトウェアでビジネスオペレーションを自動化します">}}

新しいWebテクノロジーとフレームワークの出現により、リモートビジネスの設定が不可欠になりました。顧客志向のビジネスは、Webの存在なしに関連性を維持することはできません。さらに、最近のコロナウイルスのパンデミックは、誰もがそれについて考えることを余儀なくされました。したがって、リモートまたは**ペーパーレスビジネスプロセス**をセットアップすることがこれまで以上に不可欠です。多くの現実的な戦略があります。ただし、** free **および**オープンソースソフトウェア**を使用するために、オンラインでビジネスプロセスを向上させることが最も効果的な戦略です。有能なオープンソースソフトウェアの適切なセットは、ビジネスを指数関数的に成長させることができます。
したがって、堅牢でストレスの少ない、完全で、重要なビジネスソリューションを提供します。 [Products.Containerize.com] [1]に、最も人気のある成功したエンタープライズレベルのオープンソースソフトウェアのリストを開始しました。さらに、ビジネス機能または運用の性質に応じてリストを分類しました。各オープンソースソフトウェアに関する必要な情報が提供されます。たとえば、簡単な概要、関連する** github publicリポジトリ**リンク。さらに、システムの要件、機能、インストール命令、および関連する追加の詳細も文書化します。
この**入門ブログ投稿**は、現在のCoantienize.comのカテゴリにリストされているオープンソースソフトウェア製品の概要を提供するためです。その後、リストは継続的な変更と追加の下にあります。そのため、新しい訪問ごとにカテゴリに新しい人気のあるオープンソースソフトウェアを見つけることが期待できます。別の重要なビジネス機能をカバーする新しいカテゴリを見つけることができます。

## **ビジネスウェブサイトのコンテンツを生成および整理する** {#cms}
Webサイトホーム、製品ホーム、製品ランディングページは、製品またはサービスビジネスの主要なマーケティング場所です。したがって、顧客が視聴し、試して買い物をすることができる工場のアウトレットと見なすことができます。したがって、適切なコンテンツ管理プラットフォーム、別名CMSソフトウェアが必要でした。さらに、これらのソフトウェアは、内容の構造、レイアウト、および流れを制御するのに役立ちます。 CMSソフトウェアカテゴリには、さまざまな言語やプラットフォームに多くの一般的なオープンソースソフトウェアオプションがあります。この時点で、containerize.comは次の人気のあるCMSプラットフォームをカバーしています。
  * [Joomla] [14]  - 数千の拡張機能を備えた主要なCMSソフトウェア。
  * [Drupal] [15]  - 活気に満ちたコミュニティを備えた使いやすい柔軟なPHP CMSソフトウェア
  * [Pyro] [16]  -  Laravel PHPベースの強力なコンテンツ管理システム
  * [Django CMS] [17]  -  Python DjangoベースのエンタープライズレベルCMS
  * [Umbraco] [18]  - 柔軟でフレンドリーな.NETベースのコンテンツ管理システム
より多くのCMSオプションに関する情報をすぐに作成し、有用なリソースを作成します。したがって、さらなる更新については、[コンテンツ管理] [19]セクションに注目してください。

## **コミュニティビルディングのためのディスカッションフォーラム** {#forums}
ディスカッションフォーラムは、製品、サービス、またはサービスに関する公開ディスカッションを実施するために使用されます。このような公開討論は、聴衆が議論に参加することを奨励します。したがって、コミュニティビルディングの追加の利点で、ビジネスへの洞察を開発できます。とりわけ、売上のほとんどは口コミを中心に展開しています。したがって、あなたは良いコミュニティを構築することができ、売り上げの成長も得られます。さらに、検索エンジンの観点から、コミュニティの議論は新鮮なコンテンツを生成し続けています。したがって、有機的な交通価値も追加されます。現時点では、containerize.comは以下のスタックを維持しています**ディスカッションフォーラム：**
  * [談話] [26]  - 組み込みの節度機能を備えた最高のディスカッションプラットフォーム
  * [flarum] [27]  - 豊富なユーザーエクスペリエンスを備えた軽量コミュニティディスカッションプラットフォーム
  * [バニラ] [28]  - フォーラムのカテゴリの費用対効果とクラウドベースのソフトウェアと知識ベース
  * [nodebb] [29]  -  node.jsベースのディスカッションフォーラムソフトウェアソフトウェア
  * [PHPBB] [30]  -  PHPベースのフラットスタイルの強力なディスカッションフォーラムソフトウェア
これらのオープンソースソフトウェアは、アクティブな**ディスカッションプラットフォーム**の完全なビジネスサポートを提供します。さらに、有用で人気のあるオープンソースディスカッションフォーラムソフトウェアは、[ディスカッションフォーラム] [31]セクションに必要かつ有用な情報を含むセクションに追加されます。

## **カスタマーサポート用のヘルプデスクチケットシステム** {#helpdesk}
カスタマーサポートサービスは、顧客満足度にとって最も重要な部分です。顧客満足度がなければ、特にいくつかの選択肢がある場合、あなたの生存は危険にさらされています。チケットベースのヘルプデスクシステムにより、顧客は直面している問題のプライベートチケットを作成できますが、ビジネスオーナーは顧客の問題を体系的に追跡して、ボールがドロップされないようにします。
このカテゴリでは、containerize.comは次のオープンソースの詳細を提供します**チケットシステム：**
  * [Osticket] [32]  - 広く使用されているマルチユーザーWebベースのチケットシステム
  * [uvdesk] [33]  -  eコマースにリンクされた高度にカスタマイズ可能なヘルプデスクソリューション
  * [Zammad] [34]  -  TwitterやFacebookにリンクされたマルチ言語サポートを備えた人気のあるチケットシステム
  * [freescout] [35]  - カスタマーサポートと共有受信トレイを提供するオープンソースチケットシステム
  * [helpy] [36]  -  mailgunと統合されたマルチチャネルヘルプデスクソフトウェア
これらは、顧客が問題を報告できるようにする人気のあるオープンソースサポートチケットシステムです。さらに、これらのシステムにより、サポートチームは報告された問題を解決できます。そのため、顧客とサポートチームは、報告された問題を追跡できます。さらなる更新については、[helpdeskソフトウェア] [37]セクションについてお知らせください。

  * [ビルを殺す] [78]  - 強力な無料のオープンソースの請求書および支払いソフトウェア。

これらの**金融ソリューション**は、時間節約の自動請求書、支払いの包括的な報告などを提供します。最もクールな部分は、ビジネス内でシステム全体をセットアップできる自己ホスティングのオプションです。よりオープンソースの請求書ソフトウェアが追加されます。定期的な更新については、[請求ソフトウェア] [80]セクションにアクセスしてください。

## **集中認証またはシングルサインオン** {#SSO}
**シングルサインオン**ソリューションにより、ユーザーは同じ一連の資格情報を使用して、組織内の異なるシステムにアクセスできます。エンタープライズビジネスは、潜在的な脅威とリスクに対処するために、常にセキュリティ機能を改善する傾向があります。一方、すべてのWebアプリに対して単一のユーザー管理ポイントを維持します。 containerize.comは、次のオープンソースに光を当てます** SSOソリューション：**
  * [IdentityServer] [81]  -  OpenID Connectプロトコルを使用して独自のIDプロバイダーソリューションをセットアップするための提供を提供します。
  * [keycloak] [15]  -  OAUTH2.0、OpenID Connectプロトコルを使用して、IDプロバイダーおよびアクセス管理システムを開発でき、LDAPおよびActive Directoryでうまく機能します。
  * [CAS] [82]  - 委任された認証を備えた多言語のシングルサインが点灯します。
  * [Authelia] [83]  - 非常にスケーラブルなリッチがSSOとリソース管理を特徴としていました。
  * [WSO2] [84]  - 無料のオープンソースWSO2は、すべてのID標準をサポートして認証を提供します。
  * [OpenIdentityPlatform] [85]  - シングルサインオンのオープンソースプラットフォーム、SAML 2.0 OAuth 2.0、およびOpenID Connectプロトコル。
同様に、他のカテゴリ[containerize.com] [1]は、このカテゴリに一貫して作業して、スタック内のさらにSSO製品をプッシュしています。したがって、最新の更新については、[シングルサインオン] [86]セクションについて最新情報を入手してください。
