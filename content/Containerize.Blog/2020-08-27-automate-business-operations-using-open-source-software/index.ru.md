  * [KeyCloak][15] - позволяет разработать поставщика идентификаций и системы управления доступом, используя OAUTH2.0, протоколы OpenID Connect и хорошо работает с LDAP и Active Directory.
  * [CAS][82] - многоязычный единый знак с делегированной аутентификацией.
  * [Authelia][83] - очень масштабируемый Rich представлен SSO и управление ресурсами.
  * [WSO2][84] - бесплатный открытый исходный код WSO2 поддерживает все стандарты идентификации для обеспечения аутентификации.
  * [OpenidentityPlatform][85] - Платформа с открытым исходным кодом для единого знака, SAML 2.0 OAuth 2.0 и протоколы OpenID Connect.
