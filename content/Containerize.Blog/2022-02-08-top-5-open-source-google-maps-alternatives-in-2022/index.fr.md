## OpenStreetMap {#OpenStreetMap}
[OpenStreetMap][9] est l'un des logiciels de navigation gratuits qui sont [Google Maps][3] Alternatives. Cette application ** MAP ** est gratuite, open source et est livrée avec des capacités de navigation GPS. Il est modifiable car il est livré avec un éditeur intégré. Il existe une communauté dynamique qui fournit activement le soutien. De plus, ce ** logiciel de navigation gratuit ** offre une navigation très précise et complète afin que vous puissiez repérer des endroits tels que les postes de police, etc. De plus, il vous permet d'intégrer et de télécharger un emplacement spécifique afin que vous puissiez le partager avec d'autres. Tout comme [Google Maps][3], vous pouvez obtenir des itinéraires en définissant votre source, votre destination et votre mode de voyage tels que la voiture, la marche ou le vélo. Cependant, OpenStreetMap est écrit en JavaScript avec toutes les documents concernant le développement et le déploiement. Par conséquent, vous pouvez trouver ses fichiers de code source sur [github][10].
Voici les caractéristiques importantes:
  * Hostomé
  * Navigation GPS
  * Éditeur intégré
  * Recherche
  * Partage de l'emplacement

## Osmand {#Osmand}
[Osmand][11] est une autre ** application de carte ** qui fonctionne en ligne ainsi que hors ligne. Ce ** logiciel de navigation gratuit ** est open source, personnalisable et suit l'emplacement GPS. Osmand est l'une des alternatives [Google Maps][3] qui offre une optimisation des routes, aide les utilisateurs à indiquer des chemins tels que les forêts, les paysages, etc. l'interface utilisateur. Surtout, vous pouvez planifier votre voyage selon les itinéraires suggérés par Osmand car il vous permet de rechercher divers endroits. La meilleure chose à propos de cette application de cartes hors ligne est qu'elle offre un tas de plugins qui améliorent sa portée. Osmand est principalement écrit en Java et vous pouvez trouver la documentation. Par conséquent, les développeurs peuvent trouver les fichiers de code source sur [github][12].
Osmand a les caractéristiques clés suivantes:
  * Version hors ligne
  * Texte pour parler
  * Mode nuit
  * Navigation GPS
  * Plugins

## Maps organiques {#organic}
[Maps organiques][13] est une autre application open source ** MAP ** qui fonctionne également hors ligne. Tout comme les autres logiciels Interactive Maps, il est disponible pour Android et iOS. De plus, il est livré avec les capacités de recherche les plus rapides en mode hors ligne. Organic Maps est l'une de ces alternatives [Google Maps][3] qui fournissent le mode nocturne dans lequel les thèmes deviennent sombres la nuit. De plus, vous pouvez planifier votre voyage, soit la piste de randonnée, le vélo ou par marche et il fournit également des conseils vocaux des utilisateurs. De plus, il est sans publicité, sans notifications push et aucune enregistrement requise. Organic Maps est écrit en C ++ et est livré avec la documentation. Par conséquent, vous pouvez trouver les fichiers de code source sur [github][14].
Organic Maps offre les caractéristiques importantes suivantes:
  * Fonctionne hors ligne
  * Options de recherche rapide
  * Mode sombre
  * Gratuit et open source
  * Versions Andriod & iOS

## leaflet {#leaflet}
Le [dépliant][15] est un autre logiciel de cartes interactifs open source qui est livrée avec des capacités d'auto-hébergement. Cette application ** MAP ** est légère et facile à utiliser. Surtout, il est disponible pour toutes les plates-formes de bureau et mobiles telles que Android, iOS et Web. De plus, cette application MAPS Offline Maps est très extensible et propose une API bien documentée pour les intégrations tierces. Cet open source ** Logiciel de navigation gratuit ** fournit également des didacticiels pour les conseils de l'utilisateur. La brochure est écrite en JavaScript avec une documentation complète disponible. Par conséquent, vous pouvez trouver des fichiers de code source sur [github][16].
Le dépliant propose les caractéristiques clés suivantes:
  * Hostomé
  * Poids léger
  * Support multiplateforme
  * Plugins
  * API

## Qwant Map {#qwant}
Enfin et surtout, [QWANT MAP][17] est l'une de ces alternatives ** Google Maps ** qui est open source et gratuite. Tout comme Google Maps, ce logiciel Interactive Maps offre de riches capacités de recherche. De plus, vous pouvez rechercher des endroits tels que des restaurants, des hôtels, des marchés et plus encore. De plus, il vous permet de rechercher des endroits à proximité en suivant votre emplacement. QWANT MAP est basé sur OpenStreetMap et propose un suivi GPS.
Qwant Map offre les points clés suivants:
  * Suivi GPS
  * Open source
  * Recherche globale
  * Zoom avant Zoom arrière
  * Lieux à proximité
