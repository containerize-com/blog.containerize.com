## Pengenalan singkat Vanilla Forum {#intro}
** [Vanilla Forum] [5] ** adalah perangkat lunak forum populer,. Ini adalah open source gratis dan dibangun di atas kerangka kebun. Ini menawarkan fitur -fitur kaya seperti obrolan pribadi, lampiran, beberapa plugin, dan kustomisasi yang dapat disesuaikan. Perangkat lunak forum ini adalah alat dukungan langsung ** gratis ** untuk membangun kepercayaan diri dengan pelanggan dengan memiliki komunikasi langsung dengan mereka. Lebih lanjut, integrasi forum ini dengan WordPress dapat memberi Anda penyediaan basis pengetahuan. Pengguna dapat menyimpan artikel informasi penting dan relevan untuk masyarakat dan anggota staf.
Umpan balik pelanggan selalu penting dalam hal pengembangan dan promosi produk. Namun, forum diskusi ini mendukung kolaborasi komunitas yang efektif. Ini sangat dapat dikonfigurasi dan memberikan integrasi dengan banyak perangkat lunak seperti Zendesk, WordPress, GitHub, dan banyak lagi. Selain itu, alat pendukung langsung ini mudah untuk menjadi tuan rumah mandiri dan dilengkapi dengan persyaratan sederhana untuk dipasang seperti PHP dan MySQL. Ada dokumentasi komprehensif tentang pengembangan dan penyebaran. Oleh karena itu, pengguna dapat memperoleh semua file kode sumber di [GitHub] [11].

## WordPress Insights {#WordPress}
** [WordPress] [6] ** adalah pembangunan situs web yang banyak digunakan dan platform blogging open source. Ada sejumlah besar situs web langsung yang berjalan di WordPress. Ini menyediakan semua fitur yang diperlukan dengan sejumlah besar plugin karena ada banyak plugin tingkat perusahaan. Di atas segalanya, perangkat lunak open source ini memiliki komunitas pengembang besar yang mengembangkan tema dan plugin baru. Munculnya platform blogging semacam itu telah melahirkan peluang baru bagi semua orang yang terkait dengan bisnis online. Apakah itu pengembang atau pemilik bisnis, platform ini menawarkan opsi pembangunan situs yang fleksibel.
Mudah diatur dan dilengkapi dengan dokumentasi lengkap tentang pengembangan dan penyebaran. Perangkat lunak blog ini multibahasa dan memberikan dukungan untuk lebih dari 70 bahasa. WordPress juga menyediakan integrasi dengan banyak aplikasi lain seperti [Perangkat Lunak Otomasi Pemasaran] [12], forum diskusi, dan banyak lagi.

## Cara Menginstal Forum Vanilla di WordPress {#install}
Di bagian ini, kita akan melihat cara menginstal plugin di WordPress dengan menginstal perangkat lunak forum gratis (** Forum Vanilla **) di WordPress.
Pertama, Anda dapat mengunduh plugin Forum Vanilla dari [tautan] ini [13].
Setelah plugin diunduh, tempatkan folder vanilla-forum ke/wp-content/plugin/.
Setelah itu, Anda akan dapat melihat forum vanilla di bagian bawah panel sisi kiri seperti yang ditunjukkan pada gambar di bawah ini.

{{< figure align=center src="images/Screenshot-2021-01-11-at-18.24.15-1024x548.png" alt=" Integrasi WordPress">}}

Jadi, klik pada plugin dan Anda akan mendarat ke halaman konfigurasi.

{{< figure align=center src="images/Screenshot-2021-01-11-at-18.29.14-1024x485.png" alt="Cara menginstal plugin di wordpress">}}

Selain itu, Anda dapat menyematkan forum Anda di halaman WordPress apa pun setelah konfigurasi yang berhasil menggunakan kode seperti yang ditunjukkan pada gambar di bawah ini.

{{< figure align=center src="images/Screenshot-2021-01-11-at-18.33.03-1-1024x516.png" alt="Integrasi Forum">}}

Demikian pula, jika Anda bertanya -tanya cara menginstal plugin di WordPress atau cara mengembangkan atau menyesuaikan tema apa pun, Anda dapat mengikuti artikel ini [panduan pemula untuk mengembangkan plugin WordPress] [14], [panduan pemula untuk membuat tema WordPress dari awal ] [15]. Ada banyak posting blog bermanfaat lainnya yang dapat Anda temukan [di sini] [16].
