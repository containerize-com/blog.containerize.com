## openshot {#openShot}
[OpenShot] [9] ist ein plattformübergreifender Open-Source-Video-Editor. Es bietet viele leistungsstarke Funktionen wie Trimmen, Schneiden, Effekte, 2D -3D -Animationen und benutzerfreundliche Oberfläche. Darüber hinaus ist es die beste Plattform, um Videos zu erstellen, da es REST -API- und Desktop -Integrationen bietet, auf denen Sie Ihre Dateien ziehen und fallen lassen können. Diese Open -Source -Software ist mehrsprachig, robust und sicher. Darüber hinaus gibt es umfassende Dokumentationen bezüglich Bereitstellung und Entwicklung. OpenShot ist in Python geschrieben. Daher ist der gesamte Quellcode unter [Github] [10] verfügbar.
Im Folgenden finden Sie die herausragenden Merkmale:
  * RESTful -Schnittstelle
  * Mehrere Videoeffekte
  * Timeline Vorschau
  * Sicher
  * Proxy -Bearbeitung
[Erfahren Sie mehr über OpenShot] [11]

## shotcut {#shotcut}
[Shotcut] [12] ist ein weiterer kostenloser Open -Source -Videohersteller mit vielen leistungsstarken Funktionen. Es ist mehrfach plattformübergreifend und anpassbar. Darüber hinaus sind in dieser Videobearbeitungssoftware hochwertige 360 ​​° -Videofilter und Effekte verfügbar. Der Shotcut -Videohersteller bietet Unterstützung für alle beliebten Bildformate wie TIFF, BMP, GIF, TGA, Webp, PNG, JPEG, SVG und JPEG. Shotcut kann als tragbare App aus einem externen Laufwerk ausgeführt werden. Diese Software ist jedoch in C ++, QML und JavaScript geschrieben und der gesamte Quellcode ist unter [Github] [13] verfügbar.
Shotcut verfügt über die folgenden Schlüsselmerkmale:
  * 3-Punkte-Bearbeitung
  * Hardwareunterstützung
  * Tongenerator
  * Mehrsprachig
  * Webcam Capture
  * Videoqualitätsmessung
[Erfahren Sie mehr über Shotcut] [14]

## Flowblade {#flowblade}
[Flowblade] [15] ist ein kompetenter Multitrack Open -Source -Video -Ersteller. Es erfolgt umfangreiche Funktionen zur Videobearbeitung, die eine Drag & Drop -Benutzeroberfläche enthält. Darüber hinaus verwendet dieser Video -Editor Open Source Image Filtering Technology G’Mic. Es gibt leistungsstarke Steuerelemente wie Audio -Synchronisierungen der Timeline, teilweise Teil der Videoclips, das Erstellen von komponierten Bildern und den vollständigen Track -Kompositionsmodus. Flowblade ist in Python mit ein wenig Eingabe von anderen Sprachen wie HTML, JavaScript CSS geschrieben. Daher ist sein Quellcode unter [Github] [16] verfügbar.
Flowblade bietet die folgenden wichtigen Funktionen:
  * Blend & Transformation
  * Reiche Medienkontrollen
  * Farbeffekte
  * Konfigurierbar
  * Drag & Drop UI
[Erfahren Sie mehr über Flowblade] [17]

## Kdenlive {#kdenlive}
Kdenlive ist eine weitere Open -Source -Videobearbeitungssoftware. Diese Software ist auf [Qt] [18], FFMPEG und den [KDE] [19] -Bibliotheken aufgebaut, wodurch sie alle weit verbreiteten Video-/Audio -Formate verarbeiten können. Vor allem zieht die konfigurierbare Benutzeroberfläche Benutzer an, sie für professionelle Videobearbeitungszwecke zu verwenden. Darüber hinaus gibt es umfangreiche Optionen in Bezug auf Effekte, Übergänge und Unterstützung für Audiometer und Histogramm. Darüber hinaus ist das coolste an dieser Open -Source -Software die Versionierung und automatische Sicherung von Projektdateien. Kdenlive ist in C ++ geschrieben und der gesamte Quellcode ist unter [Github] [20] verfügbar.
Kdenlive erstellt die folgenden Schlüsselmerkmale:
  * Konfigurierbare Schnittstelle
  * Proxy -Bearbeitung
  * Projektversioning
  * Unterstützung für Audio-/Videoformate
  * Multi-Track-Videobearbeitung
[Erfahren Sie mehr über Kdenlive] [21]

## Blender {#blender}
[Blender] [22] ist eine weitere Open -Source -Video -Editor -Anwendung. Es ist nicht nur ein Video -Editor, sondern auch eine vollständige 3D -Erstellungssuite, die 2D -Animation, Simulation und Bewegungsverfolgung enthält. Darüber hinaus bietet dieser Open -Source -Video -Editor eine vollständig anpassbare Benutzeroberfläche. Es gibt auch API für jede Art von Skript- und Modifikation. Blender bietet vor allem bis zu 32 Tools für die Videobearbeitung mit Mediensteuerungen. Diese Software ist in C, C ++ und Python geschrieben. Der Quellcode dieser Anwendung ist unter [Github] [23] verfügbar.
Blender bietet die folgenden wichtigen Punkte:
  * Anpassbare Schnittstelle
  * Rendern
  * Animationen
  * Live Vorschau
  * Histogramm zeigt
[Erfahren Sie mehr über Blender] [22]
