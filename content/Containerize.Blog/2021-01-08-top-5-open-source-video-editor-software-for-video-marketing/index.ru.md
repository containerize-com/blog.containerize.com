## Openshot {#openshot}

[Openshot][9]-это кроссплатформенный и отмеченный наградами видео-редактор. Он предоставляет много мощных функций, таких как обрезка, нарезка, эффекты, 2D 3D -анимация и удобный интерфейс. Кроме того, это лучшая платформа для создания видео, поскольку она предоставляет API REST и интеграции настольных компьютеров, где вы можете перетаскивать свои файлы. Это программное обеспечение с открытым исходным кодом является многоязычным, надежным и безопасным. Кроме того, существует комплексная документация в отношении развертывания и разработки. Openshot написан на Python. Поэтому весь исходный код доступен в [GitHub][10].
Ниже приведены выдающиеся функции:
  * RESTful Interface
  * Несколько видео эффектов
  * Предварительный просмотр временной шкалы
  * Безопасный
  * Редактирование прокси
[Узнайте больше об Openshot][11]

## Дробовик {#Дробовик}

[Shotcut][12] - еще один бесплатный видеопроизводитель с открытым исходным кодом со многими мощными возможностями. Это многоплатформенно и настраивается. Кроме того, в этом программном обеспечении для редактирования видео доступны высококачественные видеофильтры и эффекты 360 °. Shotcut Video Maker обеспечивает поддержку всех популярных форматов изображений, таких как TIFF, BMP, GIF, TGA, WEBP, PNG, JPEG, SVG и JPEG. Shotcut может работать как портативное приложение с внешнего диска. Тем не менее, это программное обеспечение записано в C ++, QML и JavaScript, и весь исходный код доступен в [GitHub][13].
Shotcut имеет следующие ключевые функции:
  * Редактирование с 3 очками
  * Аппаратная поддержка
  * Генератор тона
  * Многоязычный
  * Захват веб -камеры
  * Измерение качества видео
[Узнайте больше о дробовике][14]

## Flowblade {#Flowblade}

[Flowblade][15] является компетентным создателем видео с открытым исходным кодом. Он создает богатые функции редактирования видео, которые включают пользовательский интерфейс Drag & Drop. Кроме того, этот видео редактор использует технологию фильтрации изображений с открытым исходным кодом G'MIC. Существуют мощные элементы управления, такие как синхронизация аудио -шкалы, разделенная часть видеоклипов, создание композиционных изображений и режим композиции полного трека. Flowblade написана на Python с небольшим входом из других языков, таких как HTML, JavaScript CSS. Следовательно, его исходный код доступен в [GitHub][16].
Flowblade предлагает следующие важные функции:
  * Смешивание и трансформация
  * Богатые средства управления СМИ
  * Цветовые эффекты
  * Настраивается
  * Trag & Drop UI
[Узнайте больше о Flowblade][17]

## Kdenlive {#Kdenlive}

Kdenlive - еще одно программное обеспечение для редактирования видео с открытым исходным кодом. Это программное обеспечение построено на вершине [QT][18], FFMPEG и библиотек [KDE][19], из -за которых он может обрабатывать все широко используемые видео/аудио форматы. Прежде всего, его настраиваемый пользовательский интерфейс привлекает пользователей для его использования для профессионального редактирования видео. Кроме того, существуют богатые варианты, касающиеся эффектов, переходов и поддержки аудиометра и гистограммы. Кроме того, самая крутая вещь в этом программном обеспечении с открытым исходным кодом - это управление версией и автоматическое резервное копирование файлов проекта. Kdenlive записана в C ++, а весь исходный код доступен в [Github][20].
Kdenlive предлагает следующие ключевые функции:
  * Настраиваемый интерфейс
  * Редактирование прокси
  * Версия проекта
  * Поддержка аудио/видео форматов
  * Редактирование видео с несколькими треками
[Узнайте больше о Kdenlive][21]

## Блендер {#Блендер}

[Blender][22] - еще одно приложение для видео редактора с открытым исходным кодом. Это не только видео редактор, но и полный 3D -набор для создания, который включает 2D -анимацию, симуляцию и отслеживание движения. Кроме того, этот видео редактор с открытым исходным кодом предоставляет полностью настраиваемый пользовательский интерфейс. Существует также API, доступный для любого вида сценариев и модификации. Прежде всего, Blender предлагает до 32 инструментов для редактирования видео с помощью средств массовой информации. Это программное обеспечение написано в C, C ++ и Python. Исходный код этого приложения доступен в [GitHub][23].
Blender предлагает следующие ключевые моменты:
  * Настраиваемый интерфейс
  * Рендеринг
  * Анимация
  * Живой предварительный просмотр
  * Гистограмма отображается
[Узнайте больше о блендере][22]
