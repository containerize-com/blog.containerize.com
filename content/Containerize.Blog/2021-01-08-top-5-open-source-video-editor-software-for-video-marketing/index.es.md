## openshot {#openshot}
[OpenShot][9] es un editor de video multiplataforma y galardonado de código abierto. Proporciona muchas características potentes, como recorte, corte, efectos, animaciones en 3D 2D e interfaz fácil de usar. Además, es la mejor plataforma para hacer videos, ya que proporciona integraciones REST API y de escritorio donde puede arrastrar y soltar sus archivos. Este software de código abierto es multilingüe, robusto y seguro. Además, hay documentación integral disponible con respecto a la implementación y el desarrollo. OpenShot está escrito en Python. Por lo tanto, todo el código fuente está disponible en [GitHub][10].
Las siguientes son las características destacadas:
  * Interfaz RESTful
  * Múltiples efectos de video
  * Vista previa de la línea de tiempo
  * Seguro
  * Edición proxy
[Aprenda más sobre OpenShot][11]

## shotcut {#shotcut}
[Shotcut][12] es otro creador de videos de código abierto gratuito con muchas capacidades potentes. Es multiplataforma y personalizable. Además, hay filtros y efectos de video de alta calidad de 360 ​​° disponibles en este software de edición de video. Shotcut Video Maker proporciona soporte para todos los formatos de imagen populares como TIFF, BMP, GIF, TGA, WebP, PNG, JPEG, SVG y JPEG. Shotcut puede ejecutarse como una aplicación portátil desde una unidad externa. Sin embargo, este software está escrito en C ++, QML y JavaScript y todo el código fuente está disponible en [GitHub][13].
Shotcut tiene las siguientes características clave:
  * Edición de 3 puntos
  * Soporte de hardware
  * Generador de tonos
  * Multilingüe
  * Captura de cámaras web
  * Medición de calidad de video
[Aprenda más sobre el escolta][14]

## Flowblade {#flowblade}
[Flowblade][15] es un creador de video de código abierto multipista competente. Se le ocurre características de edición de video ricas que incluyen una interfaz de usuario de arrastrar y soltar. Además, este editor de video utiliza la tecnología de filtrado de imágenes de código abierto G’Mic. Existen controles potentes, como la sincronización de audio de la línea de tiempo, la parte dividida de los videoclips, la creación de imágenes compuestas y el modo de composición de pista completa. Flowblade está escrito en Python con una pequeña entrada de otros idiomas como HTML, JavaScript CSS. Por lo tanto, su código fuente está disponible en [GitHub][16].
Flowblade ofrece las siguientes características importantes:
  * Mezclar y transformación
  * Controles de medios ricos
  * Efectos de color
  * Configurable
  * Arrastre y suelta ui
[Aprenda más sobre Flowblade][17]

## kdenlive {#kdenlive}
Kdenlive es otro software de edición de video de código abierto. Este software está construido sobre [Qt][18], FFMPEG y las bibliotecas [KDE][19] debido a las cuales puede procesar todos los formatos de video/audio ampliamente utilizados. Sobre todo, su interfaz de usuario configurable atrae a los usuarios a usarla para fines de edición de video profesional. Además, hay opciones ricas con respecto a los efectos, las transiciones y el soporte para audiómetro e histograma. Además, lo mejor de este software de código abierto es la verificación de versiones y la copia de seguridad automática de los archivos del proyecto. Kdenlive está escrito en C ++ y todo el código fuente está disponible en [Github][20].
Kdenlive presenta las siguientes características clave:
  * Interfaz configurable
  * Edición proxy
  * Versiones de proyecto
  * Soporte para formatos de audio/video
  * Edición de videos múltiples
[Aprenda más sobre Kdenlive][21]

## Blender {#blender}
[Blender][22] es otra aplicación de editor de video de código abierto. No es solo un editor de video, sino una suite de creación 3D completa que incluye animación 2D, simulación y seguimiento de movimiento. Además, este editor de video de código abierto proporciona una interfaz de usuario totalmente personalizable. También hay API disponible para cualquier tipo de secuencia de comandos y modificación. Sobre todo, Blender ofrece hasta 32 herramientas para la edición de video con controles de medios. Este software está escrito en C, C ++ y Python. El código fuente de esta aplicación está disponible en [GitHub][23].
Blender ofrece los siguientes puntos clave:
  * Interfaz personalizable
  * Representación
  * Animaciones
  * Vista previa en vivo
  * Muestras de histograma
