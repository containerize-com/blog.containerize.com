## ¿Qué es Pencil2D? {#what-is-pencil2d}

[Pencil2d][7] es una aplicación de animación tradicional que permite a los usuarios crear bocetos 2D de dibujos animados y animaciones para diferentes proyectos. Puede usar Pencil2D tanto para proyectos personales como para esfuerzos comerciales, ya que es un software de animación de código abierto con licencia bajo la Licencia Pública General de GNU. Usar Pencil2D es bastante sencillo, como verá más adelante en la publicación. También puede ser utilizado por los animadores para principiantes y los usuarios profesionales.
Con [Pencil2D][7] sus animaciones se almacenan en dibujos de mapa de bits o vectores y utiliza un flujo de trabajo de ráster/vector para una gestión efectiva de animaciones. Algunos de los formatos de archivo aceptables mientras usan Pencil2D son PNG, JPEG, BMP, TIFF, etc. para imágenes y AVI, MP4, GIF y más, para videos. También es compatible con Windows, Mac OS, Linux y FreeBSD.

## ¿Cómo instalar Pencil2D? {#install-pencil2d}

Abra el terminal y actualice el índice de paquetes de su servidor local escribiendo el siguiente comando:

sudo apt-get update -y

```
Ejecute el siguiente comando en la terminal para instalar Pencil2D:

sudo apt-get install -y pencil2d

```

## Descripción general de la interfaz de aplicación {#application-interface}

Antes de comenzar con el tutorial Pencil2D real sobre cómo crear animaciones con Pencil2D, aquí hay un poco de descripción general de la interfaz de usuario, y algunas herramientas y características buenas para conocer:
* **Tablero blanco:**  Cuando lanza Pencil2D por primera vez, verá un tablero blanco principal en el medio, rodeado de paneles, herramientas y características de los controles donde creará la parte principal de la animación
* **Herramientas:**  En el lado izquierdo encontrará la función Herramientas que incluye herramienta de cepillo, lápiz, borrador, herramienta de cubo para agregar color y todas las características que le permitirán esbozar una caricatura o animación y definirla.
* **Cuadro de opciones:**  debajo del cuadro Herramientas tiene la capacidad de incluir o eliminar la ventana de opciones donde obtiene la opción de definir los atributos de las herramientas que está utilizando. Por ejemplo, si está utilizando un pincel, puede alternar entre el grosor del pincel, cuán claro u oscuro aparecerá el color y características similares.
* **Caja de visualización:**  Debajo de la ventana Opciones está el cuadro de visualización donde puede manipular el boceto que ha dibujado en la placa blanca, como voltearla derecha o izquierda, aplicando la piel de cebolla a bocetos anteriores y más.
* **Paleta de colores:**  Aquí encontrará todos los colores disponibles que se pueden agregar a sus dibujos.
* **Caja de color:**  Justo debajo de la paleta de colores está la caja de color que le permite elegir tonos de color específicos de la rueda de colores.
* **Panel de línea de tiempo:**  Esta es una de las partes más importantes del Pencil2D que le permite aplicar capas a su animación que comienza con la capa de mapa de bits, la capa vectorial y la capa de la cámara. También puede agregar marcos clave para crear una animación de video, eliminar y duplicar marcos, y más.
Una vez que haya dominado y entendido estas herramientas, puede comenzar a crear animaciones y bocetos de inmediato, o tratar de aprender a medida que avanza.
