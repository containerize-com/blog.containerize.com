## Quoi de neuf dans GPT-4? {#Whats-New-in-GPT-4}

Pour récapituler, GPT (transformateur pré-formé générateur) est un modèle de traitement du langage naturel soutenu par [OpenAI][7]. En fait, GPT-3 est un réseau neuronal artificiel formé sur des ensembles de données massifs et peut mener des conversations humaines. La semaine dernière (c'est-à-dire le 14 mars), OpenAI a lancé GPT-4 qui est la prochaine version de GPT-3. Étonnamment, il y a 100 billions de paramètres dans GPT-4 qui peuvent réellement fonctionner comme un cerveau humain. En d'autres termes, c'est une simulation du cerveau humain qui peut même ressentir les émotions impliquées dans la conversation et peut prédire la phrase suivante.
Cependant, vous pouvez créer des applications de niveau d'entreprise qui peuvent effectuer des conversations bidirectionnelles sous forme de texte et des images pour couvrir les scénarios du monde réel. Eh bien, vous pouvez dire que GPT-4 est une version raffinée de GPT-3 car la dernière version a un meilleur traitement d'image, peut créer des jeux vidéo, etc. De plus, vous pouvez visiter le [site] officiel [8] pour explorer davantage.

## Comment installer le plugin WordPress pour chatgpt? {#How-to-install-WordPress-plugin-for-ChatGPT}

Cette section démontre Chatgpt pour WordPress et la procédure d'installation. Avant d'aller de l'avant, vous pouvez visiter notre catégorie [blogging][22] pour en savoir plus sur les articles écrits sur WordPress. De plus, vous trouverez comment installer WordPress sur votre serveur ainsi que des guides d'utilisation.
Maintenant, commençons **CHATGPT WordPress Intégration.**
Tout d'abord, accédez au tableau de bord WordPress et cliquez sur l'option «Plugins» que vous verrez sur le menubar gauche de la fenêtre.
Cliquez sur "Ajouter un nouveau" et tapez "chatppt" dans la barre de recherche, vous trouverez le plugin Chatgpt en haut dans les résultats de recherche comme indiqué dans l'image ci-dessous:

{{< figure align=center src="images/chatgpt-plugin.gif" alt="Plugin chatppt">}}

Maintenant, appuyez sur le bouton "Installer maintenant" pour l'installer, puis l'activer en appuyant sur le bouton "Activer". Lors de l'installation et de l'activation réussies, il est temps de configurer ce plugin **wordpress pour chatgpt.**

## Configuration du plugin ChatGpt pour WordPress {#Configuring-ChatGPT-plugin-for-WordPress}

C'est le point où vous devez avoir une clé API **Openai.**
**Veuillez visiter ce [tutoriel][9] pour apprendre à obtenir cette clé API au cas où vous ne connaissez pas la procédure.**
Ensuite, accédez aux paramètres de ce plugin Chatgpt et appuyez sur le bouton "Paramètres" et insérez votre touche API OpenAI comme indiqué dans l'image ci-dessous:

{{< figure align=center src="images/chatgpt-wordpress-integration.png" alt="CHATGPT WordPress Intégration">}}

Après cela, vous pouvez modifier le paramètre plus loin en entrant dans l'option «Chatbot». Il vous permet de modifier le mode de communication, vous pouvez définir le message de salutation et vous pouvez également définir la position de ce chatbot basé sur **AI**. Au bas de cette fenêtre, vous pouvez voir la section «shortcode». Ici, vous pouvez obtenir le shortcode que vous pouvez utiliser pour rendre l'interface Chatgpt n'importe où sur votre site Web. De plus, vous pouvez vérifier l'option pour injecter ce chatbot sur tout votre site Web. Vous pouvez voir ces options dans l'image ci-dessous:

{{< figure align=center src="images/wordpress-plugin-for-chatgpt.png" alt="Chatppt pour wordpress">}}

Une fois tout terminé, vous pouvez voir l'interface utilisateur de ChatGPT sur votre site Web.

{{< figure align=center src="images/chatgpt-for-wordpress.gif" alt="Chatppt pour wordpress">}}
