## Humhub {#Humhub}

[Humhub][8] é um software de rede social de código aberto. É amplamente utilizado e alguns com recursos de auto-hospedagem. Essa plataforma colaborativa gratuita é altamente extensível e oferece temas e módulos ricos. Ele permite que os usuários criem espaços públicos ou privados para qualquer uso, como qualquer espaço base de tarefas específico ou qualquer outro. Além disso, os membros podem aproveitar quase todas as ações de mídia social, como podem comentar, como, seguir ou postar qualquer coisa. Além disso, eles podem gerenciar seus perfis e realizar uma pesquisa global. Além disso, esta ferramenta de rede social oferece uma versão on -line, onde qualquer pessoa pode experimentar essa ferramenta gratuita. A interface do usuário do HumHub é muito lógica e os usuários podem navegar facilmente por vários elementos. Está escrito no PHP e fácil de configurar em servidores. Você pode encontrar o código -fonte do Humhub no [Github][9].
A seguir estão os recursos proeminentes:
  * Stream de atividade
  * Painel rico
  * Suporte para arquivos de mídia
  * Auto-hospedado
  * Ferramentas sociais
[ **Saiba mais sobre Humhub** ][10]

## Elgg {#Elgg}

[Elgg][11] é outra plataforma de rede social de código aberto premiado alternativo. É altamente orientado à privacidade e está sendo feito usando técnicas criptográficas. Além disso, também é muito extensível e expõe uma interface de programação de aplicativos (API) para a integração com aplicativos de terceiros. Além disso, existem muitos plugins disponíveis que estendem o escopo deste software de rede social. Essa plataforma colaborativa gratuita oferece uma ampla gama de recursos, como um sistema de notificação, um sistema em cache de controle de acesso estático de acesso, armazenamento de arquivos, visualizações e muito mais. A interface do usuário desta plataforma de rede social de código aberto é muito intuitivo. Além disso, também está escrito no PHP, toda a documentação está disponível em relação ao desenvolvimento e implantação. Portanto, você pode encontrar o código -fonte de Elgg no [github][12].
Elgg tem os seguintes recursos importantes:
  * Plugins
  * Segurança baseada em criptografia
  * API documentada
  * Auto-hospedado
  * Autenticação de usuário
**[Saiba mais sobre Elgg][13]**

## Diáspora {#Diáspora}

[Diáspora][14] também é um dos softwares de redes sociais mais amplamente utilizados. É uma rede social descentralizada e atua como uma alternativa ao Facebook. Essa plataforma colaborativa gratuita vem com recursos sociais de nível corporativo que incluem menções, tags, emojis, reações, anexos, grupos baseados em intenções e muito mais. Além disso, a diáspora é altamente extensível e oferece integração com as principais plataformas sociais classificadas, como Twitter e Tumblr. Esta rede social descentralizada é escrita em Ruby, Ruby on Rails, juntamente com algumas contribuições de outros idiomas, como o JavaScript. Os usuários podem ter documentação abrangente sobre desenvolvimento e implantação. Portanto, você pode encontrar o código -fonte da diáspora no [GitHub][15].
A diáspora oferece os seguintes recursos importantes:
  * Descentralização
  * Integração com o Twitter e Tumblr
  * Orientado a privacidade
  * Menciona, tags, emojis, reações
  * Auto-hospedado
**[Saiba mais sobre a diáspora][16]**

## Rede Social de código aberto {#Open}

[Rede Social de código aberto][17] é outra ferramenta de rede social que também vem com recursos auto-hospedeiros e extensíveis. É multilíngue e oferece suporte em vários idiomas. Além disso, esta plataforma de rede social de código aberto fornece segurança e integração de dados com bibliotecas de terceiros. Assim como outros softwares, essa ferramenta de rede social também oferece alguns recursos interessantes, como menção, tags, sistema de notificação, sistema de bloqueio, sistema de puxão, convidando amigos e muito mais. Além disso, também oferece um feed de notícias, emojis e apoio ao Captcha. Este software de rede social está escrito no PHP com um pouco de entrada do CSS. Portanto, os usuários podem aproveitar a fonte no [Github][18], juntamente com a documentação sobre desenvolvimento e implantação.
A rede social de código aberto cria os seguintes recursos principais:
  * Convide amigos
  * Sistema de notificação
  * Auto-hospedado
  * Multilíngue
  * Feed de notícias em tempo real
[ **Saiba mais sobre a rede social de código aberto** ][19]

## Tribo {#Tribo}

Por último, mas não menos importante, [Tribo][20] é outra plataforma de redes sociais que vem com comunicação baseada em threads e muito mais. Essa ferramenta de rede social oferece integração interna do Google Analytics para rastrear e analisar os dados dos usuários. A tribo está escrita em JavaScript e vem com documentação abrangente sobre desenvolvimento e implantação. Além disso, é seguro, orientado a privacidade, fácil de usar e oferece uma interface de usuário lógica. Portanto, você pode obter o código -fonte dessa ferramenta social gratuita no [GitHub][21].
Tribo oferece os seguintes pontos -chave:
  * Tags, notificação
  * Recursos de pesquisa
  * Auto-hospedado
  * Google Analytics Integrated
  * Espaços, categorização de conteúdo
[ **Saiba mais sobre a tribo** ][22]
