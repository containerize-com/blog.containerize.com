## Humhub {#Humhub}

[Humhub][8] es un software de redes sociales de código abierto. Es ampliamente utilizado y algunos con capacidades de autohospedación. Esta plataforma de colaboración gratuita es altamente extensible y ofrece temas y módulos ricos. Permite a los usuarios crear espacios públicos o privados para cualquier uso, como cualquier espacio de base de tareas en particular o cualquier otro. Además, los miembros pueden aprovechar casi todas las acciones de las redes sociales, como pueden comentar, como, seguir o publicar cualquier cosa. Además, pueden administrar sus perfiles y pueden realizar una búsqueda global. Además, esta herramienta de red social ofrece una versión en línea donde cualquiera puede experimentar esta herramienta gratuita. La interfaz de usuario de Humhub es muy lógica y los usuarios pueden navegar a través de varios elementos fácilmente. Está escrito en PHP y fácil de configurar en servidores. Puede encontrar el código fuente de Humhub en [GitHub][9].
Las siguientes son las características destacadas:
  * Últimas acciones
  * Panel rico
  * Soporte para archivos multimedia
  * Autohostado
  * Herramientas sociales
[ **Aprenda más sobre Humhub** ][10]

## Cajón {#Cajón}

[Elgg][11] es otra plataforma de redes sociales de código abierto alternativa galardonada. Está altamente orientado a la privacidad y se realiza utilizando técnicas criptográficas. Además, también es muy extensible y expone una interfaz de programación de aplicaciones (API) para la integración con aplicaciones de terceros. Además, hay muchos complementos disponibles que extienden el alcance de este software de redes sociales. Esta plataforma de colaboración gratuita ofrece una amplia gama de características, como un sistema de notificación, un sistema almacenable de control de acceso de activos estáticos, almacenamiento de archivos, vistas y más. La interfaz de usuario de esta plataforma de red social de código abierto es muy intuitiva. Además, también está escrito en PHP, toda la documentación está disponible con respecto al desarrollo y la implementación. Por lo tanto, puede encontrar el código fuente de ELGG en [GitHub][12].
Elgg tiene las siguientes características clave:
  * Complementos
  * Seguridad basada en la criptografía
  * API documentada
  * Autohostado
  * Autenticacion de usuario
**[Aprenda más sobre Elgg][13]**

## Diáspora {#Diáspora}

[Diáspora][14] es también uno de los software de redes sociales más utilizados. Es una red social descentralizada y actúa como una alternativa a Facebook. Esta plataforma de colaboración gratuita viene con características sociales de nivel empresarial que incluyen menciones, etiquetas, emojis, reacciones, archivos adjuntos, grupos basados ​​en la intención y muchos más. Además, la diáspora es altamente extensible y ofrece integración con plataformas sociales de mejor clasificación como Twitter y Tumblr. Esta red social descentralizada está escrita en Ruby, Ruby on Rails junto con algunos aportes de otros idiomas como JavaScript. Los usuarios pueden tener documentación integral con respecto al desarrollo y la implementación. Por lo tanto, puede encontrar el código fuente de la diáspora en [Github][15].
Diáspora ofrece las siguientes características importantes:
  * Descentralización
  * Integración con Twitter y Tumblr
  * Orientado a la privacidad
  * Menciona, etiquetas, emojis, reacciones
  * Autohostado
**[Aprenda más sobre la diáspora][16]**

## Red social de código abierto {#Open}

[Network Social Social de código abierto][17] es otra herramienta de red social que también viene con capacidades autohostantes y extensibles. Es multilingüe y ofrece soporte en muchos idiomas. Además, esta plataforma de red social de código abierto proporciona seguridad de datos e integración con bibliotecas de terceros. Al igual que otro software, esta herramienta de red social también ofrece algunas características interesantes, como mención, etiquetas, sistema de notificación, sistema de bloque, sistema de poke, invitando a amigos y más. Además, también ofrece una noticia, emojis y apoyo para Captcha. Este software de redes sociales está escrito en PHP con un pequeño aporte de CSS. Por lo tanto, los usuarios pueden aprovechar la fuente en [Github][18] junto con la documentación con respecto al desarrollo y la implementación.
La red social de código abierto presenta las siguientes características clave:
  * Invitar a amigos
  * Sistema de notificación
  * Autohostado
  * Multilingüe
  * Feed de noticias en tiempo real
[ **Aprenda más sobre la red social de código abierto** ][19]

## Tribu {#Tribu}

Por último, pero no menos importante, [tribu][20] es otra plataforma de red social que viene con comunicación basada en hilos y mucho más. Esta herramienta de red social ofrece integración integrada de Google Analytics para rastrear y analizar los datos de los usuarios. La tribu está escrita en JavaScript y viene con documentación integral con respecto al desarrollo y la implementación. Además, es seguro, orientado a la privacidad, fácil de usar y ofrece una interfaz lógica de usuario. Por lo tanto, puede obtener el código fuente de esta herramienta social gratuita en [GitHub][21].
La tribu ofrece los siguientes puntos clave:
  * Etiquetas, notificación
  * Características de búsqueda
  * Autohostado
  * Google Analytics integrado
  * Espacios, categorización de contenido
[ **Aprenda más sobre la tribu** ][22]
