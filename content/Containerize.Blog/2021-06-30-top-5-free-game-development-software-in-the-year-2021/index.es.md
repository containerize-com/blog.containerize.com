## GDEVOLED {#GDEVOLEVE}
** [GDEVROVED][7] ** es un motor de juego de código abierto. Ofrece características robustas de prototipos junto con comportamientos preconstruidos. Esta ** herramienta de desarrollo de juegos gratuito ** permite a los usuarios crear juegos para casi todas las plataformas populares como Android, iOS, Web y Desktop. Además, los desarrolladores pueden crear juegos 2D utilizando flujos de trabajo intuitivos. Tiene una comunidad de desarrolladores sólido que ha desarrollado tantas extensiones para la parte posterior y la parte delantera. Además, esta herramienta gratuita ofrece soporte para efectos de sonido, pantallas táctiles, multitáctil, PlayStation y Xbox. Es seguro y fácil de integrar con bibliotecas de terceros. Gdevelop utiliza Piskel para diseñar, admob para mostrar anuncios y JFXR para el sonido y este diseñador de juegos proporciona animaciones ricas y efectos visuales como efectos CRT, brillo, mapa de color, aberración cromática y más. Sin embargo, GDEVelovel se escribe en C ++, JavaScript, TypeScript y todo su código fuente está disponible en [GitHub][8] junto con la documentación con respecto a la implementación y el desarrollo.
Las siguientes son las características destacadas:
  * Integraciones
  * Vista previa en vivo multiplataforma
  * Soporte para Xbox, PlayStation, Admob
  * Piskel para diseñar
  * Jfxr para efectos de sonido

## Godot {#Godot}
** [GODOT][10] ** es otro código abierto ** fabricante de juegos 3D **. Es autohospedado, liviano, robusto y construido sobre una arquitectura basada en nodos. Proporciona una interfaz RESTful para aplicaciones de terceros. Este diseñador de juegos 3D ofrece animaciones ricas, efectos de sonido y un editor en vivo. Hay nodos que los usuarios pueden conectarse de acuerdo con los requisitos de su escena. Además, es amigable para los desarrolladores y los desarrolladores de juegos pueden crear complementos para la funcionalidad necesaria. Este software gratuito para hacer juegos es multiplataforma y se ejecuta en casi todas las plataformas populares como Linux, MacOS y Windows. Sin embargo, Godot está escrito en C ++, JavaScript, TypeScript y todo su código fuente está disponible en [GitHub][11] junto con la documentación sobre la implementación y el desarrollo.
Godot tiene las siguientes características clave:
  * 2D, 3D Soporte
  * Opciones de exportación
  * Extensible
  * Scripting visual
  * Arrastre y suelta ui
** [Aprenda más sobre Godot][12] **

## SuperPowers {#SuperPowers}
** [Superpoders][13] ** es un software gratuito de juego de código abierto para crear ** juegos 2D y 3D **. Es multiplataforma, seguro y fácil de extender. Ofrece sprites sin licencia, informes de errores en vivo y resaltado de sintaxis. Este fabricante de juegos 3D permite a los desarrolladores escribir la lógica de negocios en TypeScript. Sobre todo, este editor de juegos gratuitos también viene con una versión fuera de línea. Además, existe una disposición para invitar a otros desarrolladores a trabajar en proyectos compartidos. ## panda3d {#panda3d}
** [Panda3d][16] ** es otro código abierto ** Motor de juegos 3D **. Los desarrolladores de juegos pueden construir juegos 3D en tiempo real utilizando ricas simulaciones y visualizaciones. Además, proporciona muchas características potentes, como objetivos de renderizado múltiples, prototipos rápidos, OpenGL y manejo de activos flexibles. Es autohospedado, multiplataforma, extensible y proporciona herramientas de línea de comandos para administrar activos. Sin embargo, Panda3D está escrito en C ++, Python y todo su código fuente está disponible en [GitHub][17] junto con la documentación sobre la implementación y el desarrollo.
Panda3d presenta las siguientes características clave:
  * Profiles de rendimiento
  * Portabilidad de la plataforma
  * Manejo de activos
  * Soporte para DirectX, OpenGL
  * Prototipos rápidos

## PlayCanvas {#playcanvas}
** [PlayCanvas][19] ** es otro código abierto GRATIS ** HTML5 Game Creator **. Se basa en WebGL y ayuda a administrar anuncios de medios ricos. Además, este motor de juegos basado en la web proporciona soporte para navegadores basados ​​en la web mediante el empleo de concatenación de script, minificación, carga diferida de activos no esenciales y más. Sin embargo, Panda3D está escrito en JavaScript y todo su código fuente está disponible en [GitHub][20] junto con la documentación sobre la implementación y el desarrollo.
PlayCanvas ofrece los siguientes puntos clave:
  * Optimizado para dispositivos móviles
  * Soporte para el navegador móvil/de escritorio
  * Soporte de WebGL
  * Colaboración en tiempo real
  * Desarrollo en el dispositivo
