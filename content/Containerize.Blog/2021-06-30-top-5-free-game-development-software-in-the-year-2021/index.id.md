## **Ringkasan** 
Industri video game berkembang dengan berlalunya waktu. Bidang pengembangan ini berisi sejarah panjang. Ini telah melalui banyak evolusi dan revolusi. Baru-baru ini, industri ini mendapatkan ledakan ketika jumlah gamer menjadi sangat tinggi. Ini melibatkan dua faktor. Pertama, orang hanya memiliki kegiatan dalam ruangan karena pandemi Covid-19. Kedua, sangat mudah untuk memasang game baru karena kemajuan teknologi. Namun, jumlah pengguna game telah membawa permintaan besar untuk video game di pasar. Sekarang, ada peluang baru untuk pengembang game****untuk memperkenalkan tren terbaru. Kami menerbitkan posting blog tentang [bagaimana industri game global memanfaatkan perangkat lunak open source][1] di masa lalu yang berbicara lebih lanjut tentang dampak perangkat lunak open source pada industri game. ## GDevelop {#GDevelop}

**[GDevelop][7] **adalah mesin game open source. Ini menawarkan fitur prototyping yang kuat bersama dengan perilaku yang sudah dibangun. Alat pengembangan game gratis** ini**memungkinkan pengguna membuat game untuk hampir semua platform populer seperti Android, iOS, Web, dan Desktop. Selain itu, pengembang dapat membuat game 2D menggunakan alur kerja intuitif. Ini memiliki komunitas pengembang yang kuat yang telah mengembangkan begitu banyak ekstensi untuk bagian belakang dan depan. Selain itu, alat gratis ini menawarkan dukungan untuk efek suara, layar sentuh, multi-sentuh, PlayStation, dan Xbox. Aman dan mudah diintegrasikan dengan perpustakaan pihak ketiga. GDevelow menggunakan Piskel untuk merancang, admob untuk menunjukkan iklan, dan JFXR untuk suara dan perancang game ini memberikan animasi yang kaya, dan efek visual seperti efek CRT, cahaya, peta warna, penyimpangan kromatik, dan banyak lagi. Namun, GDevelop ditulis dalam C ++, JavaScript, naskah dan semua kode sumbernya tersedia di [GitHub][8] bersama dengan dokumentasi mengenai penyebaran dan pengembangan.
Berikut ini adalah fitur yang menonjol:
  * Integrasi
  * Pratinjau Langsung Multi-Platform
  * Dukungan untuk Xbox, PlayStation, Admob
  * Piskel untuk merancang
  * Jfxr untuk efek suara
**[Pelajari lebih lanjut tentang GDevelop][9]**

## Godot {#Godot}

**[Godot][10] **adalah sumber terbuka lain** pembuat game 3D**. Ini adalah swadaya, ringan, kuat, dan dibangun di atas arsitektur berbasis node. Ini menyediakan antarmuka yang tenang untuk aplikasi pihak ketiga. Desainer game 3D ini menawarkan animasi yang kaya, efek suara, dan editor langsung. Ada node yang dapat dihubungkan pengguna sesuai dengan persyaratan adegan mereka. Selain itu, ini ramah pengembang dan pengembang game dapat membuat plugin untuk fungsionalitas yang dibutuhkan. Perangkat lunak pembuatan permainan gratis ini adalah cross-platform dan berjalan di hampir semua platform populer seperti Linux, MacOS, dan Windows. Namun, Godot ditulis dalam C ++, JavaScript, naskah dan semua kode sumbernya tersedia di [GitHub][11] bersama dengan dokumentasi mengenai penyebaran dan pengembangan.
Godot memiliki fitur utama berikut:
  * 2d, dukungan 3D
  * Opsi Ekspor
  * Dapat diperluas
  * Scripting visual
  * Seret & lepas UI
**[Pelajari lebih lanjut tentang Godot][12]**

## Kekuatan super {#Kekuatan super}

**[Superpowers][13] **adalah perangkat lunak game open source gratis untuk membuat** 2D & 3D game**. Ini lintas platform, aman, dan mudah diperluas. Menawarkan sprite bebas lisensi, pelaporan kesalahan langsung, dan penyorotan sintaksis. Pembuat game 3D ini memungkinkan pengembang menulis logika bisnis di TypeScript. Yang terpenting, editor game gratis ini dilengkapi dengan versi offline juga. Selain itu, ada ketentuan untuk mengundang pengembang lain untuk mengerjakan proyek bersama. Namun, Godot ditulis dalam naskah dan semua kode sumbernya tersedia di GitHub bersama dengan dokumentasi mengenai penyebaran dan pengembangan. Namun, Godot ditulis dalam naskah dan semua kode sumbernya tersedia di [GitHub][14] bersama dengan dokumentasi mengenai penyebaran dan pengembangan.
Kekuatan super menawarkan fitur -fitur penting berikut:
  * Dukungan untuk model 3D
  * Efek suara, font
  * Pelaporan Kesalahan Langsung
  * 2d, game 3D
  * Dukungan untuk Proyek Bersama
**[Pelajari lebih lanjut tentang negara adidaya][15]**

## Panda3d {#Panda3d}

**[panda3d][16] **adalah sumber terbuka lain** mesin game 3D**. Pengembang game dapat membangun game 3D real-time menggunakan simulasi dan visualisasi yang kaya. Selain itu, memberikan banyak fitur kuat seperti beberapa target render, prototipe cepat, opengl, dan penanganan aset yang fleksibel. Ini adalah swadaya, lintas platform, dapat diperluas, dan menyediakan alat baris perintah untuk mengelola aset. Namun, Panda3D ditulis dalam C ++, Python dan semua kode sumbernya tersedia di [GitHub][17] bersama dengan dokumentasi mengenai penyebaran dan pengembangan.
Panda3D muncul dengan fitur -fitur utama berikut:
  * Profil kinerja
  * Portabilitas Platform
  * Penanganan aset
  * Dukungan untuk DirectX, OpenGL
  * Prototipe cepat
[ **Pelajari lebih lanjut tentang panda3d** ][18]

## Playcanvas {#Playcanvas}

**[playcanvas][19] **adalah open source gratis** HTML5 Game Creator**. Ini dibangun di WebGL dan membantu dalam mengelola iklan media yang kaya. Lebih lanjut, mesin game berbasis web ini menyediakan dukungan untuk browser berbasis web dengan menggunakan gabungan skrip, minifikasi, pemuatan yang ditangguhkan dari aset yang tidak penting, dan banyak lagi. Namun, Panda3D ditulis dalam JavaScript dan semua kode sumbernya tersedia di [GitHub][20] bersama dengan dokumentasi mengenai penyebaran dan pengembangan.
PlayCanvas menawarkan poin -poin penting berikut:
  * Dioptimalkan seluler
  * Dukungan untuk browser seluler/desktop
  * Dukungan WebGL
  * Kolaborasi real-time
  * Pengembangan di perangkat
**[Pelajari lebih lanjut tentang PlayCanvas][21]**
