## **Visão geral** 
A indústria de videogames está florescendo com o passar do tempo. Este campo de desenvolvimento contém uma longa história. Foi passado por muitas evoluções e revoluções. Recentemente, essa indústria ganhou um boom, pois o número de jogadores foi altíssimo. Envolve dois fatores. Em primeiro lugar, as pessoas têm apenas atividades internas devido à pandemia covid-19. Em segundo lugar, é muito fácil instalar um novo jogo devido a avanços na tecnologia. No entanto, o número de usuários de jogos trouxe uma grande demanda por videogames no mercado. Agora, existem novas oportunidades para **desenvolvedores de jogos** introduzir as últimas tendências. ## Gdevelo {#Gdevelo}

**[GDEVEND][7] **é um mecanismo de jogo de código aberto. Oferece recursos robustos de prototipagem, juntamente com comportamentos pré-criados. Essa ferramenta de desenvolvimento de jogos gratuita** permite que os usuários criem jogos para quase todas as plataformas populares, como Android, iOS, Web e Desktop. Além disso, os desenvolvedores podem criar jogos 2D usando fluxos de trabalho intuitivos. Possui uma forte comunidade de desenvolvedores que desenvolveu tantas extensões para o fundo e o front end. Além disso, essa ferramenta gratuita oferece suporte para efeitos sonoros, telas sensíveis ao toque, multi-touch, PlayStation e Xbox. É seguro e fácil integrar com bibliotecas de terceiros. O GDevell usa Piskel para projetar, Admob para mostrar anúncios e JFXR para som e este designer de jogos fornece animações ricas e efeitos visuais como efeitos de CRT, brilho, mapa de cores, aberração cromática e muito mais. No entanto, o GDevelop está escrito em C ++, JavaScript, TypeScript e todo o seu código -fonte está disponível no [Github][8], juntamente com a documentação sobre a implantação e o desenvolvimento.
A seguir estão os recursos proeminentes:
  * Integrações
  * Visualização ao vivo de várias plataformas
  * Suporte ao Xbox, PlayStation, Admob
  * Piskel para projetar
  * Jfxr para efeitos sonoros

## Godot {#Godot}

**[GODOT][10] **é outro código aberto** 3D Game Maker**. É auto-hospedado, leve, robusto e construído em uma arquitetura baseada em nós. Ele fornece uma interface RESTful para aplicativos de terceiros. Este designer de jogos em 3D oferece animações ricas, efeitos sonoros e um editor ao vivo. Existem nós que os usuários podem conectar de acordo com os requisitos de cena. Além disso, é amigável para desenvolvedores e os desenvolvedores de jogos podem criar plugins para a funcionalidade necessária. Este software gratuito de fabricação de jogos é de plataforma cruzada e é executada em quase todas as plataformas populares, como Linux, MacOS e Windows. No entanto, Godot está escrito em C ++, JavaScript, Typescript e todo o seu código -fonte está disponível no [Github][11], juntamente com a documentação sobre a implantação e o desenvolvimento.
Godot tem os seguintes recursos importantes:
  * 2D, suporte 3D
  * Opções de exportação
  * Extensível
  * Script visual
  * Arrastar e soltar UI
**[Saiba mais sobre Godot][12]**

Oferece sprites sem licença, relatórios de erros ao vivo e destaque da sintaxe. Este criador de jogos em 3D permite que os desenvolvedores escrevam lógica de negócios no TypeScript. Acima de tudo, este editor de jogos gratuito também vem com uma versão offline. Além disso, há uma disposição para convidar outros desenvolvedores a trabalhar em projetos compartilhados. ## Panda3d {#Panda3d}

**[Panda3d][16] **é outro motor de código aberto** MOTOR DE JOGO 3D**. Os desenvolvedores de jogos podem criar jogos em 3D em tempo real usando simulações e visualizações ricas. Além disso, fornece muitos recursos poderosos, como vários alvos de renderização, prototipagem rápida, openGL e manuseio flexível de ativos. É auto-hospedado, plataforma cruzada, extensível e fornece ferramentas de linha de comando para gerenciar ativos. No entanto, o Panda3D é escrito em C ++, Python e todo o seu código -fonte está disponível no [Github][17], juntamente com a documentação sobre implantação e desenvolvimento.
O panda3D cria os seguintes recursos importantes:
  * Perfil de desempenho
  * Portabilidade da plataforma
  * Manuseio de ativos
  * Suporte para DirectX, OpenGL
  * Prototipagem rápida

## Playanvas {#Playanvas}

**[PlayCanvas][19] **é outro criador de jogos de código aberto** HTML5**. Ele é construído no WebGL e ajuda a gerenciar anúncios de mídia rica. Além disso, este mecanismo de jogo baseado na Web fornece suporte para navegadores baseados na Web, empregando concatenação, minificação, carregamento diferido de ativos não essenciais e muito mais. No entanto, o Panda3D é escrito em JavaScript e todo o seu código -fonte está disponível no [Github][20], juntamente com a documentação sobre implantação e desenvolvimento.
PlayCanvas oferece os seguintes pontos -chave:
  * Otimizado para dispositivos móveis
  * Suporte para navegador móvel/desktop
  * Suporte Webgl
  * Colaboração em tempo real
  * Desenvolvimento no dispositivo
