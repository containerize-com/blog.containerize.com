## A brief introduction of OpenShot {#intro}

[**OpenShot**][5] is an **open source video editing software**. It is cross-platform and compatible with all popular platforms such as Windows, macOS, and Linux. This open source **video editing software** is easy to install and comes up with a logical and user-friendly interface. In addition, there are many features that are enough to produce high-definition professional videos. This free video trimmer provides desktop interaction also where users can drag and drop files from the file system. OpenShot video editor is easy to set up with easy requirements and comes with comprehensive documentation regarding development and deployment. 

This **professional open source video editor** supports 2D, 3D animations, effects, and video subtitles. It is robust, secure, and provides a RESTful interface. Moreover, it is multilingual and offers support for 70+ languages. This **open source software** is written in Python and its source code is available on [Github][6]. 

## Features exploration of OpenShot {#features}

In the section of this OpenShot video editor tutorial, we will see some of the prominent features offered by OpenShot. First, you can download the setup file from this [link][7] and install the **video editor software** OpenShot. 

Once you open the editor, you will see the interface as shown in the image below. 

{{< figure align=center src="images/Screenshot-2020-12-27-at-21.47.18-1024x623.png" alt="video editing software tutorial">}}  

On the left-hand side, there is a project panel where you can import videos or you can simply drag & drop your videos for editing. Further, you can preview the video before editing. Users can find the options to create animations and effects in the videos. 

{{< figure align=center src="images/Screenshot-2020-12-27-at-22.00.04-1024x640.png" alt="video editing software tutorial">}}  

Then, you can add these videos to the timeline shown at the bottom. Here you can join several videos(clips) to compose a final video. You can configure several things here such as the length of the video, time, and speed of the clip. There are five tracks available by default. However, you can add as many as you need.

{{< figure align=center src="images/Screenshot-2020-12-27-at-22.04.00-1024x649.png" alt="openshot video editor">}}  

On the right-hand side, there is a window where you can play and test the changes. There are many options you can apply to the tracks such as overlapping, setting the duration and the order of the clips, trimming and cutting and more.

{{< figure align=center src="images/Screenshot-2020-12-27-at-22.25.17-1024x640.png" alt="OpenShot Video Editor Tutorial ">}}  

If you right-click on the clip in the timeline and go to the properties, a panel will open on the left-hand side that contains the advanced options related to the video effects, animations, and more.

###### Once you are done with the editing, you can export/download the video from the red button on the top options bar.

{{< figure align=center src="images/Screenshot-2020-12-27-at-23.46.53-1024x640.png" alt="openshot video editor">}}
