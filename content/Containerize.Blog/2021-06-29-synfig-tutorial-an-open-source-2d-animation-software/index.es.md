## ¿Qué es Synfig?   {#qué}
Synfig es uno de los mejores software de animación 2D que ha sido diseñado para artistas solistas o equipos pequeños para administrar sus flujos de trabajo y proyectos de animación. Como software de animación de código abierto, Synfig emplea un sistema de animación "sin tween" que no solo reduce el tiempo que lleva una animación para procesar, sino que también brinda a los animadores la capacidad de agregar en cuanto al marco de dibujo.
Cuando se trata de usar Synfig, realmente es un software de animación de procesamiento rápido y rápido que puede ser una gran adición a la oleoducto de cualquier artista que trabaje en solitario o en un equipo pequeño. Además, la comunidad Synfig es un lugar activo para encontrar no solo tutoriales sobre el uso de la aplicación de software, sino recursos para ayudarlo a encontrar soluciones en caso de que se encuentre con un obstáculo.
Puede usar fácilmente Synfig en múltiples sistemas operativos, incluidos Windows, Mac OS y Linux. Tenga en cuenta que mientras usa Synfig, su sistema debería tener:
  * Al menos 2 GB de RAM o más
  * Una CPU de doble núcleo a 2 GHz o más
Synfig se ha utilizado para animar muchos proyectos populares, incluido MoreVNA: Project Demos, la fecha límite de MoreVNA, el prólogo de detectives polares de Cyan LOS y muchos otros proyectos comerciales. Esto muestra el alcance de Synyfig, ya que también puede usarse para proyectos comerciales y personales. Ya sea que sea nuevo en el mundo de la animación o que esté avanzado en su carrera, Synfig puede hacer una buena edición para su conjunto de habilidades y ayudarlo a desarrollar una apretada comprensión de crear animaciones 2D increíbles. Si está buscando obtener más información sobre otros programas de software de animación, puede consultar otras guías y tutoriales que hemos hecho para usted.

## ¿Cómo instalar Synfig?   {#cómo instalar}
Para instalar Synfig 2D Animation Software en un dispositivo Linux, puede descargar directamente el archivo de instalador desde la página de inicio principal de Synfig. Obtendrá una opción para descargar el archivo de imagen de 32 bits o 64 bits, elija de acuerdo con su sistema.
Una vez descargado, abra su línea de comando y escriba los siguientes comandos:
```
sudo apt-get update
```
```
sudo apt-get install synfigstudio
```
Ahora debe poder acceder a Synfig Studio fácilmente.

## Descripción general de la interfaz de la aplicación   {#aplicación-interfaz}
Como hemos mencionado anteriormente, Synfig Studio es un software de animación de código abierto que permite a los usuarios crear animaciones 2D de procesamiento rápido y rápida, sin dos sin tween. Para comprender las herramientas y características, aquí hay una visión general rápida del Synfig Studio cuando comienza a usarlo.

{{< figure align=center src="images/Synfig-Tutorial-An-Open-Source-2D-Animation-Software-1-1024x649.png" alt="Synfig Tutorial Un software de animación 2D de código abierto">}}

  ***Lienzo:  **El lienzo es el patrón a cuadros gris en el medio donde puede dibujar su animación y aplicar efectos, capas, marcos y ediciones. Aquí encontrará el lienzo rodeado de botones a la derecha a **  bloquear/desbloquear  **cuadros de claves, cambiar al modo de animación ** ,  **Un pequeño triángulo negro en la esquina superior izquierda llamado caret ** ** , y el línea de tiempo en caso de que haya definido una animación distinta de cero.
  ***Herramientas:**  La caja de herramientas es uno de los principales componentes de Synfig donde encontrará todas las herramientas que necesitará para manipular objetos. Desde tamaños de pincel, colores, aplicando gradientes, borrar animaciones, aplicar textos y más, encontrará todo lo que necesita para crear animaciones y editarlas en el panel de herramientas en el lado superior izquierdo.
  ***Panel de pista de tiempo:**  Al crear una animación, el tiempo es clave, y con el panel de pista de tiempo puede ajustar los fotogramas clave, y todos los puntos de vista (los símbolos gráficos que aparecen en el panel de pista de tiempo para indicar el ajuste de cualquier parámetro) que esté involucrado.
  ***Panel de parámetros:**  Aquí encontrará todas las opciones y configuraciones detalladas para elementos activos en su animación, como color, ancho, opacidad y más. Es uno de los componentes centrales de Synfig.
Ahora pasemos a crear una animación con Synfig Studios.
