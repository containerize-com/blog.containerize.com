## Visão geral
Os usuários de grandes organizações acessam os recursos digitais sem interrupção usando o software  **SOT-ON (SSO)** . O gerenciamento de identidade e acesso usando o software **SSO**  se tornou parte integrante das operações de negócios de rotina. Nesta postagem do blog, passaremos por alguns dos seguintes 5 principais softwares de acesso único.

*  **[keycloak][2]**  
*  **[Cas][3]**  
*  **[Authelia][4]**  
*  **[WSO2][5]**

## IdentityServer {#IdentityServer}

[  **IdentityServer** ][7] é um software de sinal único gratuito. É uma estrutura de plataforma cruzada baseada no OpenId Connect e Oauth 2. Além disso, este software de código aberto fornece recursos de autenticação e autorização centrais para vários aplicativos. Ele suporta identidades federadas, múltiplos fluxos e autorização da API. Além disso, este software  **auto -hospedagem**  permite que os usuários façam login com um único conjunto de nomes de usuário/senhas em muitos aplicativos. No entanto, **IdentityServer**  está escrito em C# e todo o seu código -fonte está disponível no [Github][8], juntamente com a documentação sobre implantação e desenvolvimento.
A seguir estão os recursos proeminentes:
  * Provedor baseado em reivindicações
  * Plataforma cruzada
  * Personalização da interface do usuário
  * Controle de acesso para API
  * Sign-On /Satit-Out único
[  **Saiba mais sobre o IdentityServer**  ][9]

## Keycloak {#Keycloak}

[  **keycloak** ][10] é outro software livre baseado no OpenID Connect, OAuth2.0 e Saml2.0. Ele fornece  **recursos SSO**  em aplicativos da Web e serviços da Web. Acima de tudo, este software de código aberto fornece integrações com  **ldap**  e Active Directory. Existe uma interface lógica do usuário em que os usuários podem gerenciar funções, permissões e sessões. Além disso, esta solução gratuita **fornece bibliotecas de clientes para muitos idiomas como Java, JavaScript e C#.**  Keycloak**é escrito principalmente em Java, com pouca entrada de outros idiomas, como JavaScript e HTML. Seu código -fonte está disponível no [GitHub][11].
KeyCloak tem os seguintes recursos importantes:
  * Fornece adaptadores de clientes
  * UI fácil de usar
  * Autenticação como serviço
  * Integração com LDAP
  * Integração com o Active Directory
[  **Saiba mais sobre o keycloak**  ][12]

## Cas {#Cas}

[  **CAS** ][13] é um software SSO de código aberto com autenticação delegada. Além disso, é multilíngue e apresenta serviços de autorização central usando o protocolo baseado em ingressos. Este software livre é construído na arquitetura cliente-servidor.  **Serviço de Autenticação Central (CAS)**  Suporta muitos protocolos como OpenID, OAuth, OpenId Connect, Rest, WSFederação e SAML. Acima de tudo, existe um sistema abrangente para integração com aplicativos de terceiros. Este **SOFTWARE**  SOBRAMA**está escrito em Java. No entanto, seu código -fonte está disponível no [Github][14] com toda a documentação sobre desenvolvimento e implantação.
CAS oferece os seguintes recursos importantes:
  * Multilíngue
  * Autenticação multifator
  * Muitos mecanismos de autorização (por exemplo, ABAC, hora/data, descanso)
  * UI para gerenciar o monitoramento e as estatísticas
  * Gerenciamento de senhas
[  **Saiba mais sobre CAS**  ][15]

## Authelia {#Authelia}

[  **Authelia** ][16] é um software SSO altamente rico e escalável. Oferece ótima segurança e fornece habilidades de assinatura e login único. Este  **software de código aberto**  também fornece suporte para LDAP e Active Directory. Além disso, existe uma interface de usuário intuitiva que permite que os usuários naveguem facilmente. Além disso, fornece autenticação de dois fatores com base no  **Google Authenticator**  OTP. Este **software livre**  funciona com um proxy reverso, como o nginx. No entanto, o Authelia está escrito no idioma Go e todo o seu código -fonte está disponível no [Github][17].
Authelia cria os seguintes recursos importantes:
  * 2f Autenticação com Yubikey
  * Verificação de Identidade
  * Bloco ataques de força bruta
  * Funciona com proxies reversos
  * Notificações push com dupla
 **[Saiba mais sobre Authelia][18]** 

## WSO2 {#WSO2}

[  **WSO2** ][19] é outro sistema de gerenciamento de identidade e acesso de código aberto amplamente usado. Ele suporta quase todos os padrões de identidade populares para fornecer autenticação. Ele expôs as canecas finais da API para a integração com outros aplicativos.  **WSO2**  cria uma interface **amigável**  que é altamente personalizável. Além disso, este software de código aberto também oferece autenticação de dois fatores. No entanto, é escrito principalmente em Java e seu código -fonte está disponível no [Github][20] com documentação sobre desenvolvimento e implantação.
O WSO2 oferece os seguintes pontos -chave:
  * Alta performance
  * Integrações em nuvem
  * Flexível
  * Provisionamento de entrada/saída
  * Provedor de identidade
 **[Saiba mais sobre WSO2][21]**
