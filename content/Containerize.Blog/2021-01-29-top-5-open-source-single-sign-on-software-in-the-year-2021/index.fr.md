## Le logiciel de connexion unique offre une connexion centrale pour tous les comptes d'utilisateurs et les systèmes informatiques. Il modernise l'expérience utilisateur en offrant la sécurité et la transparence.

## Aperçu
Les utilisateurs des grandes organisations accèdent à des ressources numériques sans couture à l'aide du logiciel **Sign-On (SSO) **. La gestion de l'identité et de l'accès à l'aide du logiciel  **sso**   est devenu une partie intégrante des opérations commerciales de routine. Dans ce billet de blog, nous passerons par certains des 5 meilleurs logiciels de connexion Open Source populaires suivants.

## IdentityServer   {#IdentityServer}
[**IdentityServer **][7] est un logiciel de connexion unique gratuit open source. Il s'agit d'un cadre multiplateforme basé sur OpenID Connect et OAuth 2. De plus, ce logiciel open source fournit des capacités d'authentification et d'autorisation centrales pour plusieurs applications. Il soutient les identités fédérées, les flux multiples et l'autorisation d'API. De plus, ce  **logiciel d'auto-hébergement **  permet aux utilisateurs de se connecter avec un seul ensemble de noms d'utilisateur / mots de passe sur de nombreuses applications. Cependant,  **IdentityServer**   est écrit en C # et tout son code source est disponible sur [GitHub][8] ainsi que la documentation concernant le déploiement et le développement.
Voici les caractéristiques importantes:
  * Proviseur basé sur les réclamations
  * Multiplateforme
  * Personnalisation de l'interface utilisateur
  * Contrôle d'accès pour l'API
  * Single connexion / déconnexion
[**En savoir plus sur IdentityServer** ][9]

## Keycloak   {#KeyClOak}
[**Keycloak **][10] est un autre logiciel libre basé sur OpenID Connect, OAuth2.0 et SAML2.0. Il fournit des capacités  **sso **  entre les applications Web et les services Web. Surtout, ce logiciel open source fournit des intégrations avec  **ldap **  et Active Directory. Il existe une interface utilisateur logique où les utilisateurs peuvent gérer les rôles, les autorisations et les sessions. De plus, cette solution  **gratuite **  fournit des bibliothèques clients pour de nombreuses langues telles que Java, JavaScript et C #.  **KeyCloak**   est principalement écrit en Java avec peu d'entrée d'autres langues telles que JavaScript et HTML. Son code source est disponible sur [GitHub][11].
Keycloak a les caractéristiques clés suivantes:
  * Fournit des adaptateurs clients
  * UI convivial
  * Authentification comme service
  * Intégration avec LDAP
  * Intégration avec Active Directory
[**En savoir plus sur Keycloak** ][12]

## CAS   {#cas}
[**CAS **][13] est un logiciel SSO open source avec authentification déléguée. De plus, il est multilingue et propose des services d'autorisation centraux en utilisant le protocole basé sur les billets. Ce logiciel gratuit est construit sur l'architecture client-serveur.  **Central Authentication Service (CAS) **  prend en charge de nombreux protocoles tels que OpenID, OAuth, OpenID Connect, REST, WSFEDERATION et SAML. Surtout, il existe un système complet d'intégration avec des applications tierces. Ce logiciel  **Sign-On**   est écrit en Java. Cependant, son code source est disponible sur [GitHub][14] avec toute la documentation concernant le développement et le déploiement.
CAS offre les caractéristiques importantes suivantes:
  * Multilingue
  * Authentification multifactorielle
  * De nombreux mécanismes d'autorisation (par exemple ABAC, heure / date, repos)
  * UI pour gérer la surveillance et les statistiques
  * Gestion des mots de passe
[**En savoir plus sur Cas** ][15]

## Authelia   {#Authelia}
[**Authelia **][16] est un logiciel SSO très riche et évolutif. Il offre une grande sécurité et offre des capacités de réglementation unique et de connexion. Ce  **logiciel open source **  fournit également une prise en charge de LDAP et Active Directory. De plus, il existe une interface utilisateur intuitive qui permet aux utilisateurs de naviguer facilement. De plus, il fournit une authentification à deux facteurs basée sur  **Google Authenticator **  OTP. Ce  **logiciel libre**   fonctionne avec un proxy inversé tel que Nginx. Cependant, Authlielia est écrite en langue Go et tout son code source est disponible sur [GitHub][17].
Authelia propose les caractéristiques clés suivantes:
  * Authentification 2F avec Yubikey
  * Vérification d'identité
  * Bloquer les attaques de force brute
  * Fonctionne avec les procurations inverses
  * Notifications push avec duo
**[En savoir plus sur Authellia][18]** 

## wso2   {# wso2}
[**WSO2 **][19] est un autre système d'identité open source et d'accès largement utilisé. Il prend en charge presque toutes les normes d'identité populaires pour fournir l'authentification. Il a exposé des pintes API à la fin de l'intégration avec d'autres applications.  **WSO2 **  propose une  **interface conviviale ** **  hautement personnalisable. De plus, ce logiciel open source offre également une authentification à deux facteurs. Cependant, il est principalement écrit en Java et son code source est disponible sur [GitHub][20] avec la documentation concernant le développement et le déploiement.
WSO2 propose les points clés suivants:
  * Haute performance
  * INTESTRATIONS Cloud
  * Flexible
  * Provisioning entrant / sortant
  * Fournisseur d'identité
**[En savoir plus sur WSO2][21]**
