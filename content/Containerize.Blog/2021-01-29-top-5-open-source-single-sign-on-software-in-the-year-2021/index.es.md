## IdentityServer   {#IentityServer}
[**IdentityServer **][7] es un software de inicio de sesión único gratuito de código abierto. Es un marco multiplataforma basado en OpenID Connect y OAuth 2. Además, este software de código abierto proporciona capacidades de autenticación y autorización central para múltiples aplicaciones. Admite identidades federadas, múltiples flujos y autorización de API. Además, este  **Software de auto -alojamiento **  permite a los usuarios iniciar sesión con un solo conjunto de nombres de usuario/contraseñas en muchas aplicaciones. Sin embargo,  **IdentityServer**   está escrito en C# y todo su código fuente está disponible en [GitHub][8] junto con la documentación sobre la implementación y el desarrollo.
Las siguientes son las características destacadas:
  * Proveedor basado en reclamos
  * Multiplataforma
  * Personalización de la interfaz de usuario
  * Control de acceso para API
  * Inicio de sesión /inicio de sesión único
[**Aprenda más sobre IdentityServer** ][9]

## KeyCloak   {#KeyCloak}
[**KeyCloak **][10] es otro software gratuito que se basa en OpenID Connect, OAuth2.0 y Saml2.0. Proporciona  **Capacidades SSO **  en aplicaciones web y servicios web. Sobre todo, este software de código abierto proporciona integraciones con  **LDAP **  y Active Directory. Hay una interfaz de usuario lógica donde los usuarios pueden administrar roles, permisos y sesiones. Además, esta  **solución gratuita **  proporciona bibliotecas de clientes para muchos idiomas como Java, JavaScript y C#.  **KeyCloak**   se escribe principalmente en Java con poca entrada de otros idiomas como JavaScript y HTML. Su código fuente está disponible en [GitHub][11].
KeyCloak tiene las siguientes características clave:
  * Proporciona adaptadores de clientes
  * UI fácil de usar
  * Autenticación como servicio
  * Integración con LDAP
  * Integración con Active Directory
[**Aprenda más sobre KeyCloak** ][12]

## CAS   {#cas}
[**CAS **][13] es software SSO de código abierto con autenticación delegada. Además, es multilingüe y se les ocurre los servicios de autorización central utilizando el protocolo basado en boletos. Este software gratuito se basa en la arquitectura de cliente cliente.  **Servicio de autenticación central (CAS) **  Admite muchos protocolos como OpenID, OAuth, OpenID Connect, REST, WSFederation y SAML. Sobre todo, existe un sistema integral para la integración con aplicaciones de terceros. Este  **software **  de inicio de sesión**  está escrito en Java. Sin embargo, su código fuente está disponible en [GitHub][14] con toda la documentación con respecto al desarrollo y la implementación.
CAS ofrece las siguientes características importantes:
  * Multilingüe
  * Autenticación multifactorial
  * Muchos mecanismos de autorización (por ejemplo, ABAC, hora/fecha, descanso)
  * UI para administrar el monitoreo y las estadísticas
  * Administración de contraseñas
[**Aprenda más sobre CAS** ][15]

## Authelia   {#Authelia}
[**Authelia **][16] es un software SSO altamente rico y escalable. Ofrece una gran seguridad y ofrece habilidades de regulación de inicio de sesión e inicio de sesión. Este  **Software de código abierto **  también proporciona soporte para LDAP y Active Directory. Además, hay una interfaz de usuario intuitiva que permite a los usuarios navegar fácilmente. Además, proporciona autenticación de dos factores basada en  **Google Authenticator **  OTP. Este  **Software gratuito**   funciona con un proxy inverso como Nginx. Sin embargo, Authelia está escrita en el lenguaje GO y todo su código fuente está disponible en [Github][17].
Authelia presenta las siguientes características clave:
  * 2f Autenticación con yubikey
  * Verificación de identidad
  * Bloquear ataques de fuerza bruta
  * Funciona con proxies inversos
  * Push Notificaciones con Duo
**[Aprenda más sobre Authelia][18]** 

## WSO2   {#WSO2}
[**WSO2 **][19] es otro sistema de gestión de identidad y acceso de código abierto ampliamente utilizado. Admite casi todos los estándares de identidad populares para proporcionar autenticación. Ha expuesto Pints ​​API End para la integración con otras aplicaciones.  **WSO2 **  Viene con una interfaz  **fácil de usar**   que es altamente personalizable. Además, este software de código abierto también ofrece autenticación de dos factores. Sin embargo, se escribe principalmente en Java y su código fuente está disponible en [Github][20] con documentación sobre el desarrollo y la implementación.
WSO2 ofrece los siguientes puntos clave:
  * Alto rendimiento
  * Integraciones en la nube
  * Flexible
  * Aprovisionamiento entrante/saliente
  * Proveedor de identidad
**[Aprenda más sobre WSO2][21]**
