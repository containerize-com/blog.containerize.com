## ¿Qué es CivicRM? {#civi}

Este [Software de gestión de relaciones constituyentes][4] es de código abierto, gratuita y viene con capacidades de autohostación. CIVICRM es altamente extensible y ofrece integración con el software de gestión de contenido más utilizado, el software de blogs como [WordPress][11], [Joomla][12], [Drupal][13] y más. Este  **Software de automatización gratuito**  es de características ricas y viene con herramientas de informes para generar varios informes. Además, le permite establecer campañas de defensa, recaudación de fondos entre pares, gestión de eventos y gestión de casos.
Esta  **herramienta de automatización de marketing**  es fácil de configurar y requiere requisitos simples para configurarlo en servidores que incluyen PHP y MySQL. Sobre todo, ofrece características altamente eficientes, como búsqueda rápida, conjuntos de datos personalizados, seguimiento de interacción, registro de eventos en línea, pagos de aceptación, informes de eventos y muchos más. Además, le permite aplicar filtros en los datos, enviar regalos a amigos y familiares y ofrece una provisión para ahorrar informes en diferentes formatos. Además, CIVICRM está escrito en PHP junto con la entrada de otros idiomas como JavaScript y HTML. Por lo tanto, puede encontrar archivos de código fuente en [GitHub][14].
